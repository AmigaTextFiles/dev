Installation
Kopiere das "nano" Verzeichnis irgendwo auf deine Festplatte.

Amiga

Beispiel: nano ist in "Work:nano":

Füge die folgenden Zeilen in deine "user-startup" Datei: 

path Work:nano add
setenv nanoinc "Work:nano/include/"
setenv nanoprog "Work:nano/prog/"
setenv nanotemp "T:vm_"

PC

Beispiel: nano ist in "C:\nano":

Füge die folgenden Zeilen in deine "autoexec.bat" Datei: 

set PATH=c:\nano\;%PATH%
set nanoinc=c:\nano\include\
set nanoprog=c:\nano\prog\
set nanotemp=%TEMP%\vm_
