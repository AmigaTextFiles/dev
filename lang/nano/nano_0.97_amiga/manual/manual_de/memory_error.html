Speicher Fehlercodes
Die Standardeinstellung ist es das Programm zu beenden, wenn ein "Speicherfehler" auftritt.
Du kannst die Variable "_err_alloc" auf "1" setzen und die Fehlerbehandlung einschalten.
Jede Feld Reservierung gibt einen Code in der Variablen "_alloc" zurück.

Die Codes sind in der Includedatei "memory.nah" definiert: 
    code|   error
    ----+----------------
    0   |   no memory
    1   |   allocation ok
