Preprozessor

 
    L = long Register, D = double Register, S = string Variable

Registernamen

    #setreg_l       L, name;
    #setreg_d       D, name;


    Beispiel:

    #setreg_l       L0, x;      setzt Name "x" für Register "L0"

    push_i          5, x;

include

    #include <filename>         fügt die Datei in das Programm ein

line parser

    #setquote       S;          ändert das Anfürungszeichen in den String
    #setsepar       S;          ändert den Separator (Komma) in den String
    #setsemicol     S;          ändert das Semikolon in den String


    Die Standardeinstellungen sind:
    ein String ist umgeben von Anführungszeichen: "
    Argumente der Befehle sind durch ein Komma: , getrennt
    
    Beispiel:

    #setquote       ';
    #setsepar       |;

    print_s         '"foo", "bar"';
    print_n         L0;
    mul_l           L1 | L2 | L3;
