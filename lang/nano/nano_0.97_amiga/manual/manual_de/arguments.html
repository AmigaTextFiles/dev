Argumente

 

    L = long Register
    S = string Variable


    argnum      L;              gibt die Anzahl der Shell Argumente zurück:
                                -1 = keine Argumente, 0 = ein Argument...

    argstr      L, S;           kopiert das Argument mit dem Index "L" in einen String
