Assembler statements

An assembler statement allows you to insert assembler code right in your pascal code.


Assembler statements




More information about assembler blocks can be found in the Programmer's guideThe register list is used to indicate the registers that are modified by an assembler statement in your code. The compiler stores certain results in the registers. If you modify the registers in an assembler statement, the compiler should, sometimes, be told about it. The registers are denoted with their Intel names for the I386 processor, i.e., 'EAX', 'ESI' etc...
