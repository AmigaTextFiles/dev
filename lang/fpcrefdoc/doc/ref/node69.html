The system unit

 

The system unit contains the standard supported functions of Free Pascal. It is the same for all platforms. Basically it is the same as the system unit provided with Borland or Turbo Pascal.

Functions are listed in alphabetical order. Arguments to functions or procedures that are optional are put between square brackets.

The pre-defined constants and variables are listed in the first section. The second section contains the supported functions and procedures.
