
Required: ixemul.library 40.4 or higher.

General instructions: Click the buttons to set up the printer the way 
you want. Test by pressing the TEST button.

Load the file you want printed into the file requester and press PRINT.

MaxLineLen is an ACE program that scans a textfile and reports the 
longest line in the file. This should help you when determining the 
pitch to set.
