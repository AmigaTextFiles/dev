
LABEL OPTIONS:

* How Many Times  = Print the label this many times
* Label Size      = Large is 4" x 1 7/16" 
                    Small is 3 1/2" x 15/16"

THIS PROGRAM IS HARDWIRED TO FIVE FIELDS:
  NAME,ADDRESS,CITY,STATE,ZIP
FIELDS MUST BE SEPARATED BY COMMAS

Use the String gadgets labelled Amount, Instructions
and Date to enter data for the COD tag. Click Print COD
and the selected label will be sent to a UPS COD tag.

You MUST change the data in the script LAB.cod.scp to
reflect your own name and address.

The String gadgets for Name, Addr, etc. can be used to
Add a label to the list.
