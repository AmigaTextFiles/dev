MagicMail - Main window

MagicMail is a new e-mail-program that I am currently developing. Some of the features that the program will have:

    Groups. All mails can be divided into separate groups.
    Filters. All incoming (and outgoing) mails can be automatically divided into separate groups. You just make a filter that the mails should match, eg if the From: - field contains a specified address, the mail will be moved to a specified group.
    MIME support. MagicMail will have full MIME support. In the current ßeta version only the most common MIME commands are supported.
    MIME multipart support. MagicMail splits up multipart mails so that uuencoded parts are not viewed as text, etc.
    Queue for all mails. You don't have to be online when you write the mails. You can send all queued mails and get all new mails at once when you are online.
    Sort-function. Sort by subject / sender / date&time.
    Export mails / separate parts.
    Uses one (or one per group) file for all mails. This is much faster and much more space efficiant than having separate files for all mails.
    Online help. Uses gadutil.library to show help for all gadgets when moving over them.
    Does NOT use MUI!!!

Many of these features are already in the program, but as I haven't had the time, I haven't finished it yet.

The main window of MagicMail contains the group and message lists. The other gadgets in the window is used to switch to read mode, move, delete, send, get and send&get messages.
