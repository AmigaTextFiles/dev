 ~~~~~~~
	* Satte in en RememberKey i strukturen som allokeras för de gadgets
	  som använder hotkeys eller är någon type av GadUtil gadget. Kan
	  användas för allokering av extra data för den gadget den hör till.

	* Alla LISTVIEW_KIND gadgets kommer att allokera den extra data-
	  strukturen (behöver RememberKey för noder).

	* Satte in tre flaggor för placeringen av texten i en bevel box
	  gadget.

		- BB_TEXT_ABOVE - Standard inställning. Placerar texten med
				med start vid bevel boxens övre kant.

		- BB_TEXT_IN	- Placerar texten så att halva texten hamnar
				under boxens övre kant.

		- BB_TEXT_BELOW - Placerar texten så att hela texten hamnar
				under boxens övre kant.

	* Gjorde en ny rutin - GU_OpenFont. Den här rutinen öppnar en font
	  med diskfont.library's OpenDiskFont, om diskfont gick att öppna
	  då libraryt öppnades för första gången. Annars används gfx.library
	  för att öppna fonten. Gfx library klarar bara att ladda fonter som
	  redan är inladdade från disk och ROM fonter.

	* GU_OpenFont används internt av text_width, GU_RefreshBoxes och
	  create_layout_gad (.printtext).
