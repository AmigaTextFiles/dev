DiskDupe

A yet to be released disk duplicater. This program program works with up to 4 floppy drives at the same time. The disk is written to a disk to, or read from, an disk image file of exactly 901120 bytes. The program uses one of my own extensions to the AppWindow - AppGadgets... drag and drop filenames into the string gadgets :).
