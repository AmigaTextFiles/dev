\chapter{Prop.mui/Prop.mui}
\rule{\textwidth}{1mm}
\vspace{0.5cm}
\begin{deflist}{MMMMMMMM}
\item[\bf FUNCTION]
Prop class generates the well known proportional gadgets. It offers the same attributes as a usual boopsi gadget of propgclass. However, MUI's prop gadgets allow using any imagery for the knob and for the background.
\end{deflist}

\begin{deflist}{MMMMMMMM}
\item[\bf NAME]
\begin{description}
\item[MUIA\_Prop\_Horiz]  -- (V4 ) [I.G], BOOL
\end{description}

\item[\bf FUNCTION]
Determine if you want a horizontal or a vertical
prop gadget.

Defaults to FALSE, i.e. vertical.

\item[\bf SEE ALSO]
MUIA\_Prop\_Entries, MUIA\_Prop\_Visible, MUIA\_Prop\_First
\end{deflist}


\subsection{Prop.mui/MUIA\_Prop\_Slider}
\rule{\textwidth}{1mm}
\vspace{0.5cm}
\begin{deflist}{MMMMMMMM}
\item[\bf NAME]
\begin{description}
\item[MUIA\_Prop\_Slider]  -- (V4 ) [ISG], BOOL
\end{description}

\item[\bf FUNCTION]
Indicate that this prop gadget is used in a slider. MUI might
then use different imagery. Since you really should use the
slider class when creating sliders, you normally don't need
to care about this attribute.
\end{deflist}
