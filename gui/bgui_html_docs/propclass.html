top propclass/--background--
NAME

	Class: propclass
	Superclass: baseclass
	Include File: 

FUNCTION

	To provide a BGUI compatible extension of the system propgclass. In
	addition to the normal propgclass functionality you get the possibility
	to add arrow gadgets to the object. Objects from this class send out the
	following attribute pairs in notification events:

	GA_ID   - Gadget object ID.
	PGA_Top - Top position of the prop knob.

NOTE

	All of the system propgclass attributes are also valid for this class.
	Please refer to the documentation on the propgclass for more information
	on these attributes.


top propclass/PGA_Arrows
NAME

	PGA_Arrows -- ( BOOL )

FUNCTION

	To determine whether or not the created object will get arrow gadgets
	attached to the proportional gadget.

DEFAULT

	TRUE.

APPLICABILITY

	(I).

SEE ALSO

	PGA_ArrowSize


top propclass/PGA_ArrowSize
NAME

	PGA_ArrowSize -- ( ULONG )

FUNCTION

	To determine the size of the arrows. The size affected is the height
	of the arrows in vertical prop objects or the width of the arrows in
	horizontal prop objects.

DEFAULTS

	10 horizontal and 9 vertical.

APPLICABILITY

	(I).

SEE ALSO

	PGA_Arrows


top propclass/PGA_NoFrame
NAME

	PGA_NoFrame -- ( BOOL ) ** V40 **

FUNCTION

	To make a scroller object without a frame around the proportional
	part of the object. Any attached arrows will keep framing.

	FALSE.

	(I).
