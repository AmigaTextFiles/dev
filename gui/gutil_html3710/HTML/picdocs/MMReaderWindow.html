MagicMail - Reader window

The reader window of MagicMail contains a list for the different parts of the message, and a reader area that shows the message. The other gadgets in the window is used to switch to list mode, write a new message, reply to the displayed message, forward, resend, read next/prev message, read next/prev unread message, switch the header on/off in the message view area, and delete the current mail.
