NAME
	GU_SetToggle -- Change status of a toggle-select gadget.

   SYNOPSIS
	GU_SetToggle(status, gadget, window)
	             D0,     A0,	 A1

	VOID GU_SetToggle(BOOL, struct Gadget *, struct Window *);

   FUNCTION
	This function selects or unselects a toggle gadget, that means a
	gadget with the tag GU_ToggleSelect set to TRUE.

	For some more information about the tags, look in <libraries/gadutil.h>
	or in the autodoc for the GU_LayoutGadgetsA() function.

   INPUTS
	status - New status for the gadget. TRUE for selected, FALSE for
	         unselected.
	gadget - The gadget to change.
	window - Window that the gadget is located in.

   RESULT
	none

   BUGS
	none known

   SEE ALSO
	GU_LayoutGadgetsA(), <libraries/gatutil.h>, <libraries/gadutil.i>
