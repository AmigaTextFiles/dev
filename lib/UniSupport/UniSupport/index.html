UniSupport Library: Copyright 2008 Joerg van de Loo

This is a static library for the use with linkers and shall make it easier for you to encode/decode UTF-strings.

You should pay attention due to the fact that "UniSupportLib" is in a beta stage and with that, some bugs may surface that I've overlooked.




Miscellaneous functions:

GetMappingTable()
InitLocalHook()
UniAccuUTF8StrLen()
UniLegalUTF8Boundary()

 Transcoding functions

UniConvertToUTF8()
UniFromUTF8ToSces()


Installation:

Just copy "unisupport.h", which can be found in the "proto" drawer, to your OS include-directory's "proto" drawer.
"proto/unisupport.h" contains all information required by your C-compiler in order to be used with the static libraries.
Okay, I know that the information covered by "proto/unisupport.h" shouldn't be placed inside of the "proto" drawer, but then again, it makes installation far more easy.
Once you did it, install the needed static libraries of your choice to your compiler's lib directory.



That's all for now. I hope that these functions are of use for you and are illustrating how to make use of functions provided by UniLibrary.
