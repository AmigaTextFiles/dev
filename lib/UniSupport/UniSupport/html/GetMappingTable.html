UniSupp/GetMappingTable UniSupp/GetMappingTable
NAME
GetMappingTable
SYNOPSIS
struct MapTableSBCES * GetMappingTable( ULONG iana_id)
FUNCTION
Get the table which belongs to the used character set.
INPUTS
iana_id is an IANA-ID as found in the IANA character-sets database.
RESULT
One of the supported mapping tables or NULL if this IANA-ID is not supported.
