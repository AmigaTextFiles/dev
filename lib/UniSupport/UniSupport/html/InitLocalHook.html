UniSupp/InitLocalHook UniSupp/InitLocalHook
NAME
InitLocalHook
SYNOPSIS
void InitLocalHook( struct Hook *hook, HOOKFUNC callbackfunc, APTR data)
FUNCTION
Init the hooks accordingly to the OS we use.
INPUTS
hook is the memory location of your Hook structure.
callbackfunc is your function (routine) which gets called by the corresponding OS function.
data is any kind of information that is attached to the Hook (h->Data).
RESULT
Initialized Hook structure ready to be used.
BUGS
