NAME
UniFromUTF8ToSces
SYNOPSIS
BOOL UniFromUTF8ToSces( const TEXT *src, const ULONG inlength, const ULONG cps, TEXT **dest, ULONG *outlen, ULONG iana_id)
FUNCTION
UniFromUTF8ToSces() transcodes the input text of type UTF-8 into the equivalent singlebyte character encoding scheme by taking into account the IANA-ID.
INPUTS
src points to the text encoded in UTF-8.
inlength is the amount in bytes that src is made of.
cps is the amount code points (UTF-8 multibyte sequences) that src contains.
dest is a pointer to a 32-bit variable that contains after successful transcoding the memory location where the singlebyte encoded text can be found.
outlen is a pointer to a 32-bit variable that informs how many bytes dest contains (equal to number of characters).
iana_id is one of the IANA-IDs that specifies the type of single byte character encoding scheme, i.e. you can specify what kind of encoding scheme should be used to encode the input string - if none is set, IANA_ISO_8859_1 is assumed. For OS4 you should specify IANA_ISO_8859_15 since it is default here.
RESULT
TRUE if transcoding could be successfully performed.
If FALSE is returned you may not access dest and outlen.
BUGS
NOTES
dest will be allocated by using AllocVec()!
You have to FreeVec() it once you don't need it anymore.
