NAME
UniLegalUTF8Boundary
SYNOPSIS
const TEXT * UniLegalUTF8Boundary( const TEXT *src, const TEXT *stop)
FUNCTION
Ensure that the last character of a UTF-8 multibyte sequence ends on a legal boundary, i.e. that it is either an ordinary US-ASCII character code or the last trail octet of a UTF-8 multibyte sequence.
INPUTS
src (POINTER) is the memory location where the text starts.
stop (POINTER) points one byte behind last character code of your buffer (src + amount_characters = stop).
RESULT
Modified stop-pointer - which points now (if not already) one byte behind last valid character or points to 'src' if an error occurred.
