net/holzammer/gui/dt/dtviewfile.m

Description:

Contains a function to view any kind of file via DataTypes system in a separate window.

Requirements:

Compiler: CreativE
OS: AmigaOS >=V39
CPU: >=68020

Functions:

dtviewfile()

Purpose: View a file using DataTypes
Usage: boolean:=dtviewfile(path,mode)
Parameters: path a STRING with the complete AmigaDOS path to the file to view.
mode a CONST determining the mode. Currently there is only DTVIEWFILE_SYNC. The function will not return until the user closes the window.
Return Values: TRUE if viewing the file was successful
FALSE on failure
