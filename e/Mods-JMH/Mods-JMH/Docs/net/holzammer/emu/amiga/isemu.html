net/holzammer/emu/amiga/isemu.m

Description:

Contains a function to detect if running under Emulation

Requirements:

Compiler: CreativE
OS: AmigaOS >=V39
CPU: >=68020

Functions:

isemu()

Purpose: Checks if application is running under Emulation
Usage: boolean:=isemu()
Parameters: none
Return Values: TRUE if running under Emulation
FALSE if not running under Emulation
Notes: Currently only UAE and Amithlon are detected
