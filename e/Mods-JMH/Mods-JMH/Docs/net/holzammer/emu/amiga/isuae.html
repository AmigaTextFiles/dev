net/holzammer/emu/amiga/isuae.m

Description:

Contains a function to detect if running under UAE (UnixAmigaEmulator)

Requirements:

Compiler: CreativE
OS: AmigaOS >=V39
CPU: >=68020

Functions:

isuae()

Purpose: Checks if application is running under UAE
Usage: boolean:=isuae()
Parameters: none
Return Values: TRUE if running under UAE
FALSE if not running under UAE
