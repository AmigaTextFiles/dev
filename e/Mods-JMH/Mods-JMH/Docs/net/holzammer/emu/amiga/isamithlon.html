net/holzammer/emu/amiga/isamithlon.m

Description:

Contains a function to detect if running under Amithlon

Requirements:

Compiler: CreativE
OS: AmigaOS >=V39
CPU: >=68020

Functions:

isamithlon()

Purpose: Checks if application is running under Amithlon
Usage: boolean:=isamithlon()
Parameters: none
Return Values: TRUE if running under Amithlon
FALSE if not running under Amithlon
