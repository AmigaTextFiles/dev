Es gibt zwei Arten von Updates:

1. Minor-Updates: ist die häufigere Variante von beiden - das Programm wurde
   nicht sehr stark verbessert. Sie brauchen daher kein neues KeyFile und
   brauchen auch NICHT zu bezahlen. Wenn Sie es direkt vom Autor beziehen,
   sollten Sie immer eine Diskette sowie genügend Rückporto (DM 2,- innerhalb
   Deutschlands) beilegen (Sendungen mit zu wenig Rückporto oder ohne Diskette
   werden NICHT beantwortet !!). Sie erhalten dann die neueste Version des Pro-
   gramms. Ihr aktuelles KeyFile wird bis zum nächsten Major-Update mit ALLEN
   Minor-Updates zusammenarbeiten.
2. Major-Updates: tritt nicht sehr häufig auf - das Programm wurde sehr stark
   verbessert. Um alle Funktionen nutzen zu können, müssen Sie Ihr KeyFile up-
   daten lassen. Für dieses Update MÜSSEN Sie bezahlen. Die Preise sind aus der
   Datei »Registrierung_prt.txt« ersichtlich. Wenn Sie dieses Update direkt vom
   Autor beziehen, brauchen Sie KEINE Diskette beizulegen. Das Rückporto er-
   übrigt sich ebenfalls. Sie erhalten dann die neueste Version des Programms
   sowie ein neues KeyFile, das bis zum nächsten Major-Update mit ALLEN Minor-
   Updates zusammenarbeitet.


Hinweise zum KeyFile »MaikSchreiber.key«
----------------------------------------
Dieses KeyFile enthält die Informationen von allen meinen Programmen, die ein
KeyFile benutzen. STELLEN SIE IMMER SICHER, DASS SIE DAS JEWEILS NEUESTE KEY-
FILE INSTALLIERT HABEN !! Wenn Sie nicht sicher sind, kopieren Sie notfalls das
KeyFile aus dem letzten Programmpaket, das Sie von mir erhalten haben, in Ihr
s:-Verzeichnis. Klicken Sie dazu das Icon »InstallKeyFile.bat« an.

Sollte trotzdem die »altes KeyFile«-Meldung auftauchen, obwohl Sie wirklich das
neueste KeyFile korrekt installiert haben, kann dies nur einen bestimmten Grund
haben. Sie haben von mir irgendwann Ihr KeyFile bekommen. In der Zwischenzeit
habe ich bereits ein neues Programm geschrieben, daß das KeyFile nutzt. Sie
haben inzwischen von anderen Quellen eine neuere Version des Programms er-
halten. Diese Version berücksichtigt aber bereits das veränderte KeyFile, sodaß
es mit Ihrem alten KeyFile nicht funktioniert. Sie brauchen also nur ein neues
KeyFile. Schreiben Sie einfach an obenstehende Adresse mit der Anfrage auf ein
Minor-Update. Sie können auch auf ein Major-Update anfragen.

How to get updates
------------------
Updates can always be ordered from the author:

Maik Schreiber

FR Germany (Europe)

NOTE: you can only pay cash (put the money in the envelope) !!


There are two kinds of updates:

1. Minor update: is the more frequent variant of them - the program wasn't im-
   proved very much. So you don't need a new key file and DON'T need to pay.
   If you order directly from the author, you should add a floppy disk and
   enough return postage (DM 2.- within FR Germany) (orders with too few return
   postage or without floppy disks WILL NOT be answered !!). You will get the
   newest version of the program. Your current key file will work with ALL
   minor updates until the next major update.
2. Major update: is the less frequent variant - the program was improved very
   much. To be able to use all functions, you must update your key file. You
   MUST pay for this update. See »Registration_prt.txt« for the prices. If you
   order directly from the author, you NEED NOT add a floppy disk. The return
   postage is also superfluous. You will get the newest version of the program
   along with a new key file which will work with ALL minor updates until the
   next major update.
