PURPOSE

  Convert SFD files in E source.
  Source included.

REQUIREMENTS

  OS37+

  Must be started from Shell.
  Command line is FILE/A,CASE/S,VERSION/N'.

  FILE    : the file to be converted
  CASE    : if present then case is preserved
  VERSION : includes functions only upto the specified version
