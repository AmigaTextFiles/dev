• Description:
    This small tool shows some information about cached modules and can free it, if needed. It is very important, if You change some modules what are in cache and won't be used by d.

  • Syntax: 
      dcache F=FLUSH/A,S=SHOW/A

    Without any arguments, it shows count of cached modules and size of memory occupied by these modules. If FLUSH argument used, frees all the cached modules. If SHOW argument is used, it shows all cached modules with it's name, it's size and type. Type is ascii or binary depending on the module.
