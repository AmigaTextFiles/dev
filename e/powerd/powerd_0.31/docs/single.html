OperatorNameComment
+plusUseless, only for You :^)
-negation
~bit inversion
&addressReturns address of the variable
^on addressReturns value on address given by the variable
++post/pre additionSee below
--post/pre subtractionSee below

  • Negation: 
      a:=-4
  b:=-a                         // b=4
  • Inversion:
    Returns inversed (bit) variable 
      a:=16                         // a=$00000010
  b:=~a                         // b=$ffffffef
  b~=a                          // b=$ffffffef
  • Address:
    Returns address of the variable 
      b:=&a                         // b contains address of a

ATTENTION:
  b&=a                          // this is not an address, but Bit AND
  • On address:
    Returns long on address in the variable 
      b:=^a                         // b contains long on address in a
  • Post/Pre addition/subtraction:
    If ++ or -- are after the variable then returned value is the contain of variable, then is the (number of ++ or -- minus one)*1 added/subtracted to/from the variable. If ++ or -- are before the variable then is the (number of ++ or -- minus one)*1 added/subtracted to/from the variable and result is the returning value. 
      a:=10                         // a=10
  a--                           // a=9
  a----                         // a=6
  a++                           // a=7
  b:=a++                        // b=7, a=8
  b:=+++a                       // b=10, a=10
