h2m v1.0
by Martin Kuchinka (16.5.2000)

  • What it is:
    H2M is an utility that can convert c-header (#?.h) files to d-modules (#?.m). Source code is included in PowerD archive ('d:source/h2m.d'). You can do with it everything you like except of selling it :)

  • Limitations:
    It is very difficult to make such converter, because of macros, which can use also c-code. And this can't be converter to d-code in a simple way. When You convert a module which has such macros, You have to correct them to usable code handy. Sorry.

    Macros are slightly changed to be more d-comparible: 
      "  (doublequote)     will be converted to ' (apostrophe)
  '  (apostrophe)      will be converted to ' (doublequote)
  -> (c-object member) will be converted to . (d-object member)
  0x (c-hexdec header) will be converted to $ (d-hexdec header)
  #L (constant long)   will be converted to # where # is a hex/dec number


  • Features:
    It saves (some) comments.
    If aligns OBJECT definition comments.

  • How to use it:
    It has only one argument: 
      h2m SOURCE/A

    Where SOURCE should be a c-header file without '.h' extension. It will write file with same name, but with extension '.m'.

  • Example: 
      1.> h2m include:intuition/screens
