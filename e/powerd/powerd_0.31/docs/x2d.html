x2d v1.0
by Martin Kuchinka

  • What it is:
    It's an ultimate conversion utility, that should convert all the known file formats like the c-headers, e-modules, fd-files and much more, but currently it can convert only the fd files. You can see, that it's not very ultimate yet ;), but the FD conversion is very important, it's not just another fd2m utility, this also generates the ppc link-lib files to be able to use 68k libraries within ppc code, it's currently required to generate these files.

  • Limitations:
    It's limited more then You can even imagine ;)

  • How to use it:
    It has just one argument, the source file, what can currently be only some FD file. Nevertheless if You don't define this argument, a requester will open.
