• Popis: Urcite kazdy programator chtel alespon jednou ve svem programu pouzit jednu cast kodu vicekrat a musel to udelat zpusobem, ze proste prekopiroval danou cast zdrojoveho kodu tam, kde ho potreboval. Pomoci teto vymozenosti je toto minulosti. Staci si napsat kod, ktery potrebujete pouzit vicekrat mezi klicova slova SUB <jmeno> a ENDSUB, a pak uz jen pouzivat jmeno tohoto podprogramu.

  • Syntaxe:
    • Definice: 
        SUB <nazev>
    <podprogram>
  ENDSUB

      Kde <podprogram> je vas kod, ktery chcete, aby se spustil pri zavolani tohoto podprogramu pomoci <nazev>.
    • Pouziti: 
        <nazev>

      Pri kazdem pouziti tohoto nazvu se zavola dany podprogram a po jeho skonceni bude program pokracovat dale.


  • Poznamky:
    • azda SUB definice musi byt ulozena uvnitr dane progedury, obsah daneho podprogramu nijak neovlivni misto, kde je definovat.
    • Davejte pozor pri opusteni podprogramu drive, nez pomoci ENDSUB, protoze to muze zpusobit problemy se zasobnikem!!!


  • Ukazka: 
    PROC main()
   DEF	result
   result:=10
   print
   result*=15
   print
   result/=9
   print

   SUB print
      PrintF('result is \d.\n',result)
   ENDSUB
ENDPROC

    Tento programek vypise toto: 
    result is 10.
result is 150.
result is 16.
