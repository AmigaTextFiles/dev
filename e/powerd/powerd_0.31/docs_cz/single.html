  • Negace: 
      a:=-4
  b:=-a                         // b=4
  • Inverze:
    Vraci invertovanou promennou (bitove) 
      a:=16                         // a=$00000010
  b:=~a                         // b=$ffffffef
  b~=a                          // b=$ffffffef
  • Adres:
    Vraci adresu promenne 
      b:=&a                         // b obsahuje adresu a

POZOR:
  b&=a                          // toto neni adresa, ale bitove AND!!!
  • Na adrese:
    Vraci long na adrese dane promennou 
      b:=^a                         // b obsahuje long na adrese v a
  • Post/Pred pricteni/odecteni:
    Pokud je ++ nebo -- az za promennou, navracena hodnota bude hodnota pred timto prictenim/odectenim. Pricteni/odecteni bude aplikovano az po navraceni hodnoty. Lze pouzit vice - ci + podle potreby, minimum je ale -- ci ++, coz odecita/pricita jednicku. Kazde dalsi + nebo - zvysuje tento prirustek o jedna. Pokud je ++ nebo -- pred promennou, dojde nejdrive k pricteni/odecteni promenne, a az potom j navraceni hodnoty: 
      a:=10                         // a=10
  a--                           // a=9
  a----                         // a=6
  a++                           // a=7
  b:=a++                        // b=7, a=8
  b:=+++a                       // b=10, a=10
