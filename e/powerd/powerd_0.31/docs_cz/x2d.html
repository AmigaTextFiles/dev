  • Co je to:
    Toto je prevratna konverzni utilitka, ktera by zkonvertule vsechny zname typy souboru tak, aby se dali pouzit s PowerD, jako c hlavickove soubory, e moduly, fd soubory atd, atd. Nyni jsou podporovany pouze hlavickove soubory cecka, fd soubory a pragma/lib soubory... Takze momentalne to neni az tak prevratna konverzni utilitka ;) Ale konverze FD souboru je velmi dulezita hlavne proto, ze x2d vytvari kromne PowerD kompatibilniho modulu i potrebny ppc interface, ktery je nezbytny pri pouziti 68k knihoven v PowerPC kodu.

  • Omezeni:
    Zatim je tento programek omezen vice, nez si vubec dokazate predstavit ;)

  • Jak ho pouzivat:
    Tento program ma pouze jediny argument, jimz je zdrojovy soubor, coz muze byt nejaky z vyse uvedenych souboru. Pokud nebude zdroj zadan, otevre se requester pro jeho vyber.
