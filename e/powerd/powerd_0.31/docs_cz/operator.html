• priority operatoru:
    operatory s vyssi prioritou budou uprednostneny pred operatory s nizsi prioritou: 
      x:=1+2*3   // 2 bude nasobena 3, vysledek bude pridan 1
             //  a tento vysledek ulozen do x
  x-=1<<2*3  // 1 bude posunuta o 2 doleva, vysledek
             // bude vynasoben 3, vysledek bude odezten od x

    toto plati pouze za pouziti DPRE volby (a ta je defaultni).

  • ukazky rovnic: 
      DEF a,b,c
  a:=10                         // a=10
  b:=a\4                        // b=2
  c:=a>>2                       // c=2
  a+=b+3*c                      // a=18
  a:=:c                         // c=18, a=2
  b:=a+3+c-=12                  // c=6, b=11
