Zkompilujte si vas zdrojak do souboru '#?.o' (objekt) a napiste si pro nej externi ascii modul
  • Priklad:
    Napisme si malou funkci v Cecku: 
      int test(int a, int b)
  {
    int c;
    c=a*b+10;
    return(c);
  }

    kterou chceme volat ze zdrojaku PowerD. Pojmenujme si tento zdrojak 'test.c'. Nejdrive si ho zkompilujme ve vasem oblibenem Cecku. To vytvori objektovy soubor 'test.o', ktery obsahuje nasi funkci. Pak si vytvorime externi modul, ktery nam umozni tuto funkci vyuzivat ve zdrojacich PowerD. Pojmenujme si tento modul 'dmodules:my/test.m': 
      OPT LINK='test.o'

  LPROC test(a:LONG,b:LONG)(LONG)

    soubor 'test.o' si prekopirujte do adresare 'd:lib', nebo zapiste kompletni cestu (napr: OPT LINK='mysrc:lib/test.o'). Potom pokazde, kdyz pouzijete: MODULE 'my/test', budete moct pouzit vasi Ceckovskou funkci

    Pokud chcete tuto funkci pouzit pouze v jedinem PowerD zdrojaku, je zbytecne vytvaret externi modul, staci ty dve radky zapsat primo do zaseho PowerD zdrojaku.

    Pokud je vse v poradku, lze vasi Ceckovskou funkci pouzivat v PowerD: 
      x:=test(y,5)
