• Novinky:
    • 0.20a2
      • pridano klicove slovo SELECTED
      • jiz nezalezi na poradi klicovych slov CASE/DEFAULT/SELECTED


  • Syntaxe: 
      SELECT <exp>
  CASE <exp1>             // vice klicovych slov CASE muze byt pouzito
    <prikazy1>
  CASE <exp2>
    <prikazy2>
  ...
  [ DEFAULT
    <prikazy> ]
  [ SELECTED
    <prikazy0> ]
  ENDSELECT [ <list> ]

    <exp> je zdrojovy vyraz, ktery bude porovnavan s kazdym dalsim vyrazem <exp1> atd. <exp1> muze obsahovat vycenasobne vyrazy. (viz nize) Pokud alespon jeden z porovnanych vyrazu je stejny jako vyraz <exp>, pak <prikazy1> budou provedeny. Pokud ne, dalsi CASE bude otestovano. Pokud zadne CASE nevyhovovalo vyrazu <exp>, <prikazy> (za DEFAULT) budou provedeny. A pokud minimalne jeden z CASE vyhovoval, <prikazy0> budou provedeny po jeho skonceni.

  • Vicenasobne vyrazy:
    Jednotlive vyrazy museji byt oddeleny carkou (,): 
      CASE <a>,<b>; <prikazy>

    Pokud se vyraz <exp> rovna <a> nebo <b>, pak <prikazy> budou provedeny.

  • Rozmezi hodnot ve vyrazech:
    Vyraz <exp1> muze obsahovat take rozmezi hodnot, ktere je podmineno pouzitim klicoveho slova TO: 
      CASE <a> TO <b>; <prikazy>

    kde <a> je minimum a <b> je maximum tohoto rozmezi. Pokud se vyraz <exp> vejde do tohoto rozmezi, <prikazy> budou provedeny.

    POZNAMKA: Oboji, rozmezi i vicenasobne vyrazy mohou byt kombinovany!

  • Vraceni hodnot:
    Podivejte se predevsim na ukazky nize, hlavne na klicove slovo 'IS'.

  • Ukazky: 
      SELECT age
  CASE 0 TO 17
    PrintF('Young\n')
  CASE 18 TO 50
    PrintF('Adult\n')
  CASE 51 TO 120
    PrintF('Old\n')
  DEFAULT
    PrintF('What???\n')
  ENDSELECT

  name:=SELECT Person.ID
    CASE 1 IS 'Paul'
    CASE 2 IS 'Jenny'
    CASE 3 IS 'Peter'
    CASE 4 IS 'Mark'
    ENDSELECT 'unknown'

    Otestujte si tyto ukazky s ruznymi hodnotami a vse jiste pochopite. 
      PROC main()
    DEF a,s[16]:STRING
  
    PrintF('Enter a number 1-10: \n')
    ReadEStr(stdout,s)
    a:=Val(s)
    SELECT a
    SELECTED;     PrintF('good selection!\n')
    DEFAULT;      PrintF('bad number!\n')
    CASE 1 TO 3;  PrintF('gimme more!\n')
    CASE 4 TO 6;  PrintF('average!\n')
    CASE 7 TO 9;  PrintF('that much?\n')
    CASE 10;      PrintF('too much...\n')
    ENDSELECT
  ENDPROC
  
  PROC main()
    DEF a,s[16]:STRING
    PrintF('Enter Your age 1-100: \n')
    ReadEStr(stdout,s)
    a:=Val(s)
    PrintF('You are \s.\n',
      SELECT a
      CASE  0 TO  3 IS 'small baby'
      CASE  4 TO  6 IS 'baby'
      CASE  7 TO 10 IS 'child'
      CASE 11 TO 19 IS 'teenager'
      CASE 20 TO 50 IS 'adult'
      CASE 51 TO 99 IS 'old'
      DEFAULT       IS 'not a human ;)'
      ENDSELECT)
  ENDPROC
