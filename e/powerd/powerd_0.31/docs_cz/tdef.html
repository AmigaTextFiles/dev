• Popis:
    Klicove slove TDEF umoznuje definovat vlastni typy jako s Ceckovym typedef. Je to ponekud omezenejsi, nez Ceckovy ekvivalent, ale muze udelat tu samou praci.

  • Syntaxe: 
      TDEF   <nazev>           // <nazev> je VOID typ
  TDEF   <nazev>:<typ>    // <nazev> je <typ> typ
  TDEFL  <nazev>           // <nazev> je LONG typ
  TDEFUL <nazev>           // <nazev> je ULONG typ
  TDEFB  <nazev>           // <nazev> je BYTE typ
  TDEFUB <nazev>           // <nazev> je UBYTE typ
  TDEFW  <nazev>           // <nazev> je WORD typ
  TDEFUW <nazev>           // <nazev> je UWORD typ
  TDEFF  <nazev>           // <nazev> je FLOAT typ
  TDEFD  <nazev>           // <nazev> je DOUBLE typ



  • Ukazka:
    Pokud casto pouzivate treba 'PTR TO RastPort' typ, lze nadefinovat: 
      TDEF RP:PTR TO RastPort

    a pak uz staci jen prepsat vsechny 'PTR TO RastPort' typy kratsim 'RP'.
