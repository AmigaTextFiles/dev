• Popis:
    Pomoci techto klicovych slov je mozne predcasne opustit, ci znovu-zacit smycku. Tato klicova slova mohou byt pouziva ve LOOP, IF, WHILE, REPEAT, SELECT a FOR smyckach.

  • Nocinky:
    • Od verze 0.16
      • pridana klicova slova NEXTIF a NEXTIFN


  • Syntaxe: 
      EXITIF <vyraz> [ DO <prikazy> ] [ ( IS / GIVES / GIVING / RETURNING ) <list> ]

    Zatimco vyraz <vyraz> je FALSE nic se nedeje, pokud je TRUE, <prikazy> budou provedeny a <list> hodnot bude navracen. Lze take pouzit EXITIFN k negaci <vyraz>u.
      NEXTIF <vyraz> [ DO <prikazy> ]

    Zatimco <vyraz> je FALSE nic se nedeje, pokud je TRUE, <prikazy> se provedou smycka zacne od zacatku, to znamena, ze zbytek kodu mezi NEXTIF a ENDWHILE/ENDLOOP/atd bude preskocen, a smycka bude pokracovat. Lze take pouzit NEXTIFN k negaci <vyraz>u.

  • klicove slovo EXIT: 
      EXIT [ DO <prikazy> ] [ ( IS / GIVES / GIVING / RETURNING ) <list> ]

    Klicove slovo EXIT je vice-radkovym ekvivalentem jedno-radkoveho klicoveho slova IS. Je vceklu neuzitecne pro smycky (LOOP, WHILE, atd), ale je velmy uzitecne pro prikazy IF a SELECT. Umoznuje z techto prikazu vracen hodnoty. Kuk na vraceni hodnot: IF, SELECT.

  • Poznamka:
    Klicova slova GIVES, GIVING a RETURNING maji vsechna stejny vyznam jako IS, je to jen kvuli lepsi citelnosti kodu.
