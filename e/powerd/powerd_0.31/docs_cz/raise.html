• Popis:
    RAISE je klicove slovo, ktere umoznuje preddefinovat promenne exception a exceptioninfo, jenz budou nastaveny jako argumenty k funkci Raise(), pokud bude vysledek dane funkce splnovat dane podminky.

  • Syntax:e 
      RAISE <exc> [ ,<exci> ] IF <funkce>()<podminka><hodnota>

    Kde: <exc> a <exci> jsou hodnoty promennych exception a exceptioninfo. <funkce> je jmeno dane funkce. <podminka> je jedno z: =, <, >, =<, => nebo <>. <hodnota> je hodnota, s niz ma byt porovnan vysledek <funkce>. Pokud <hodnota> a vysledek z funkce odpovida podmince <podminka>, bude provedena vyjimka.

  • Ukazka: 
      RAISE "FILE" IF Open()=NIL,
        "MEM"  IF AllocVec()=NIL

  f:=Open('ram:file',OLDFILE)
  m:=AllocVec(256,0)

    ja toio same jako: 
      IF (f:=Open('ram:file',OLDFILE))=NIL THEN Raise("FILE")
  IF (m:=AllocVec(256,0))=NIL THEN Raise("MEM")
