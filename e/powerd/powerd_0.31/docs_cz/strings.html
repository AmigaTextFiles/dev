  • Vice nasobne retezce:
    Jediny retezec vzdy zacina a konci jedinym apostrofem. Vicenasobne retezce funguji uplne stejne s tim rozdilem, ze pokud string ukoncite apostrofem, a hned za nim nasleduje znak + (plus), ale pak pokracovat v tom samem stringu pomoci dalsiho apostrofu. Timto muzete dany string rozdelit na vice radek, nebo do nej vkladat stringy definovane v makrech. Maximalni delka takoveho stringu je omezena na 1024 znaku. Vicenasobne stringy mohou obsahoval libovolny pocet jednoduchych stringu.

  • Ukazka jednoducheho a vicenasobneho retezce: 
      'bla'
  'Hello world!\n'

  'My address:\n'+
  'Amforová 1930\n'+
  'Prague, 15500\n'+
  'Czech Repiblic\n'
  • Ukazke formatovacich retezcu (pouzijte je s funkcemi PrintF(), StringF() nebo podobnymi): 
      PrintF('a+b=\d\n',a+b)
  PrintF('file ''\s'' not found.\n',filename)
  PrintF('Address is $\z\h[8]\n',adr)

  • Ukazky rozsireneho formatovani retezcu: 
      'Date: \xDn.\xMn.\xy' vytvori neco jako 'Date: 3.10.2001'

  • Ukazka pouziti \jx: 
      'Hello\j10'           // je to same jako 'Hello\n'
  'Test \j12345'        // je to same jako 'Test {45'
  '\j999'               // je to same jako 'c9'
