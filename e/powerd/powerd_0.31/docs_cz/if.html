• syntaxe:
    syntaxe IF muze byt velice rozmanita, zde nasleduji jak jednoduche, tak vicecetne syntaxe: 
      IF <vyraz> THEN <kod> [ ELSE <kod> ]

  IF <vyraz>
    <kod>
  [ ELSEIF <vyraz>
    <kod> ]    // vicecetne ELSEIF zde muze byt
  [ ELSE
    <kod> ]
  ENDIF

  IF <vyraz>       <kod>
  [ ELSEIF <vyraz> <kod> ]    // vicecetne ELSEIF zde muze byt
  ELSE           <kod>

    vicerarkove syntaxe (dve predchozi syntaxe) se mohou kombinovat. jednoduchy kod muze byt nahrazen klicovym slovem DO. pri pouziti klicoveho slova DO za klizovym slovem ELSE se nesmi pouzit klicove slovo ENDIF: 
      IF <vyraz>     [ DO <kod> ]
  ELSEIF <vyraz> [ DO <kod> ]
  ELSE           DO <kod>

    i pri viceradkove definici lze pouzit jednoradkove ELSE, ktere zkracuje delku zdrojoveho kodu, tato definice vypousti klicove slovo ENDIF: 
      IF <vyraz>
    <kod>
  [ ELSEIF <vyraz>
    <kod> ]    // vicecetne ELSEIF zde muze byt
  ELSE <kod>

  • negovane podminky:
    pridanim 'N' za klicove slovo IF nebo ELSEIF, bude vysledek podminky negovan: 
      IF a>b
  IF a=NIL

    je to same jako: 
      IFN a<=b
  IFN a

  • vraceni hodnot: 
      a:=IF <vyraz> THEN <tvyraz> ELSE <fvyraz>

  a:=IF <vyraz>
       <kod>
       EXIT <tvyraz>
     [ ELSEIF <vyraz>
       <kod>
       EXIT <evyraz> ]    // vicecetne ELSEIF zde muze byt
     ELSE
       <kod>
       EXIT <fvyraz>
     ENDIF

  a:=IF <vyraz>       <tvyraz>
     [ ELSEIF <vyraz> <evyraz> ]    // vicecetne ELSEIF zde muze byt
     ELSE           <fvyraz>

    IF <vyraz> is TRUE, a bude obsahovat <tvyraz> jinak a bude obsahovat <fvyraz>. If ELSEIF <vyraz> is TRUE, a bude obsahovat <evyraz>. je mozne vracet i vicenadobne hodnoty. viz klicove slovo EXIT.
