collectionX.applyExistsFrom()

     NAME
          applyExistsFrom()
     SYNOPSIS
          hits := collectionX.applyExistsFrom(collectionX:PTR TO collectionX)
     FUNCTION
          Goes threw every node and checks if one with
          the corresponding index exists in other list
          if so, get its value-field.
     RESULT
          nr of hits mapwise thislist <> otherlist.
     NOTES
          this and that list MUST be sorted before calling this!
          (if its not already, eg. u have been using ordInsert())
          it's faast :)
     SEE ALSO
          xli.sort(), xli.ordInsert()
