A simple, if less than eloquant, approach is to dump the
data to a text file, to this you use the Open, Input, and Print commands,
check the help files for the exact syntax but the general code should
look something like this...

Save_Game:

Open In 1,FILE$ <- File is the name of the file you are saving to

For X=0 to Array_Length
For Y=0 to Array_Height
For Z=0 to Array_Depth
Etc. Depending on How many levels deep your array is...

Print 1,Str$( Array(x,y,z....) )
rem This will write array one line at a time into the text file FILE$

Next X : NExt Y : Next Z : etc....

Close 1 <- Don't forget this line (closes files, else get an unalidated
disk error)

Load_Game:

Open In 1,FILE$ <- File is the name of the file you are saving to

For X=0 to Array_Length
For Y=0 to Array_Height
For Z=0 to Array_Depth
Etc. Depending on How many levels deep your array is...

Junk$=Input$ 1 <- Grabs the next line of the file
Array(x,y,z....)=Val(Junk$)

Next X : NExt Y : Next Z : etc....

Anyway the code looks something like that, questions? Ask Away!
