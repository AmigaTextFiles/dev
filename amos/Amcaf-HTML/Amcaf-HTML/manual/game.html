Gameport Peripheral Support


AMCAF enables you to use an optional connected four player adapter as well as a second mouse connected to joystick port 1.

Moreover, you can read out the second fire button of a joystick or if the lowlevel.library is present even the other 5 buttons on the pad.

  • 4-Player Adapter commands
    • Build Adapter
    • =Pjoy
    • =Pjup
    • =Pjdown
    • =Pjleft
    • =Pjright
    • =Pfire
    • =Xfire
  • Second mouse commands
    • Limit Smouse
    • Smouse Speed
    • Smouse X
    • Smouse Y
    • =X Smouse
    • =Y Smouse
    • =Smouse Key
