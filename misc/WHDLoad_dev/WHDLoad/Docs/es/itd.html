ITD * Imagen A Disco

Herramienta para escribir imágenes de disco nuevamente a un dispositivo de disco.

Instalación

ITD solamente se encuentra en el archivo DEV archive, no en el archivo USR
cópielo a C: o una ubicación similar

Opciones

FILE/A,DEVICE,FORMAT/S
FILE/A nombre de la imagen de disco a escribir
DEVICE el dispositivo al cual ITD escribirá el disco
por defecto "DF0:"
ejemplo "RAD:"
FORMAT/S   si se especifica el disco será formateado

Código de retorno

0 si es exitoso, de otra forma 20.

Descripción técnica

ITD puede escribir a cualquier dispositivo de pistas de disco (trackdisk). Para evitar problemas (por ej. corrupción del disco duro) no escribe a dispositivos mas grandes que 2,000,000 bytes actualmente. No se realizará una verificación (esta herramienta es un código creado rápidamente ;-), así que Ud. esta avisado! Si la imagen es mas pequeña que el dispositivo Ud. vera un mensaje de advertencia, ITD no cambiara las pistas excedentes en este caso. Si la imagen es mas grande Ud. también vera un mensaje e ITD escribirá tanta información como quepa en el disco.
