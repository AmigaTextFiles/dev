Εξομοίωση Kickstart

Μερικές εγκαταστάσεις χρησιμοποιούν ένα λεγόμενο kickemu για να έχουν ένα ολόκληρο περιβάλλον AmigaOS για το εγκατεστημένο πρόγραμμα. Για αυτό το kickemu απαιτείται ένα αυθεντικό kickstart image. Αυτό το image μπορεί να εξαχθεί από μία πραγματική Amiga με εργαλεία όπως το GrabKick ή παρόμοια που μπορούν να βρεθούν στο aminet. Η υποστηριζόμενη έκδοση image εξαρτάται από το εγκατεστημένο πρόγραμμα. Πρέπει να βρείτε πληροφορίες για το ποιό image πρέπει να χρησιμοποιηθεί στο παρεχόμενο αρχείο ReadMe. Το image για εκδόσεις kickstart 1.2 και 1.3 μπορεί να εξαχθεί μόνο από μία αντίστοιχη A500, για την έκδοση 3.1 μπορεί να εξαχθεί από μία A500/A600/A2000, A1200 ή A4000 που πρέπει να έχει Kick3.1 ROM. Αν το image δεν μπορεί να εντοπιστεί από το WHDLoad ή το image είναι λάθος έκδοση ή αλλαγμένο με οποιοδήποτε τρόπο, το WHDLoad θα τερματιστεί με το κατάλληλο μήνυμα σφάλματος. Επιπλέον ένα αρχείο relocation με την επέκταση .RTB που να ταιριάζει με το kickstart image πρέπει να είναι παρόν. Αυτό το αρχείο relocation μπορεί να βρεθεί στο aminet στο πακέτο util/boot/skick346.lha. Και τα δύο αρχεία, το kickstart image και το αρχείο relocation πρέπει να τοποθετηθούν στο κατάλογο Devs:Kickstarts/ και πρέπει να έχουν το σωστό όνομα. ...για την εγγραφή νέων Slaves / εγκατάσταση προγραμμάτων

  • προτείνεται ένας 68030+ με MMU για τη χρήση όλων των χαρακτηριστικών του WHDLoad (ένα χροντρό μηχάνημα είναι πάντα καλή επιλογή για ανάπτυξη λογισμικού ;-))
  • γνώσεις για κώδικα μηχανής, assembler, amiga hardware ...
  • ένας assembler (πρέπει να μπορεί να βελτιστοποιεί)
  • μερικά άλλα εργαλεία θα είναι χρήσιμα (hexedit, disasm, ...)
  • μερικά ακόμα MB RAM
  • πρόσβαση σε διαφορετικά μηχανήματα για λόγους δοκιμών (68000 - 68060, OCS, AGA, GfxCard)
  • ένα hardware ή software freezer όπως τα AR3, HrtMon, TK
  • ένα κουμπί NMI για τη διακοπή εκτέλεσης προγραμμάτων ή για εισαγωγή σε software-monitor/freezer
