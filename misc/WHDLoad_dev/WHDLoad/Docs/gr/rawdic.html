RawDIC * nonstandard Disk Image Creator

Το RawDIC είναι το προτιμόμενο εργαλείο για τις εγκαταστάσεις του WHDLoad για τη δημιουργία αρχείων ή disk images από τις αυθεντικές δισκέτες με το δικό τους format. Για να μάθετε περισσότερα για το RawDIC, διαβάστε τις οδηγίες του, που μπορούν να βρεθούν στο πακέτο DEV του WHDload.
