This archive of autodoc-it MUST contain (to be freely distributable):

autodoc-it             - the main program (in amigaguide script format)
autodoc-it.info        - the icon 
getting_started.doc    - a short workthrough for installing/using
                         autodoc-it
autodoc-it.rexx        - arexx script for adding cross references to
                         includes, which do not work if at installation 
                         time include: was not assigned. Not needed, if you
                         convert autodocs correctly (with include: assigned
                         correctly!)
contents.txt           - this file
