The Preferences Window

Here you can see the Preferences Window.
Here you can set up some stuff about ReCatItPro that can ease your work while localizing.

The various fields means:

  • Language
    You should insert your preferred language here, such as Italiano or Svenska.

  • Author
    I suggest you to insert your name here :-)

  • Font
    Here you can select the default ReCatIt Pro fonts.
    NOTE: On some AmigaOS you'll need to restart the program to see the changes.

  • Show Smart Display
    Check this gadget to open the Smart Display Window.

  • AutoFill
    Checking this gadget, some (usually empty) fields in the Workspace will be automatically filled with the default values while importing a Catalog Descriptor or Catalog Translation.

  • Save in <Local>
    By checking this gadget, the catalogs will be saved in "LOCALE:Catalogs/(your language)/" instead of the current dir.

  • Safe Requesters
    By checking this gadget, some requesters will pop up before every destructive operation that could drive you to loss of data.
    TIP: You should disable this gadget only if you know exactly what you are doing. It is better one more requester than one work less.
  • Const.Len
    Here you can insert the maximum size of a constant. Set it to 0 if you don't want to set any limit to constant lengths.
