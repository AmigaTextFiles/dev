The Main Window

Please seat down and get comfy. I am introducing you the Main Window. This is the window you'll almost see and click in while using ReCatItPro. So I suggest you to study it very well.

As you can see from the image above, the Main Window is composed of some different parts:

  • In the upper zone of the window we have a cycle gadget you can use to change the text to be shown in the ListView in the center of the window.
    Avaible display options are:
    • Text
      This will show the text contained in the Text gadget field.
    • New Text
      This will show the text contained in the New Text gadget field.
    • Original
      This will show the text contained in the Orig gadget field.
    • Constant Name
      This will show the text contained in the Const gadget field.

    Also, in this zone, you'll find two gadgets:

    • Add (Shortcut: a)
      Use this gadget to add a new message in the catalog you are editing.
      NOTE: You should not add any message in a catalog you are just translating. This option is just meaningful if you are creating or editing a catalog file of a program of yours.
    • Del
      Use this gadget to delete a message in the catalog you are editing.
      NOTE: You should not delete any message in a catalog you are just translating.
      This gadget has no shortcut to avoid undesired deletions.

  • In the central part, we have just the ListView showing all our messages.
    NOTE: Messages inside the ListView are not sorted alphabetically. That's because messages have their own numeration and there's a reason why a message is in a particular place. Also, very often, in Catalog Descriptions message's constant value is set to -1 which means add 1 to the previous value.
    ReCatItPro preserves this order to assure catalog consistency.

  • In the bottom part of the Main Window, you can see all the fields that build up a message. Only three of them are actually editable, but here we'll describe all of them.
    • Num (Shorcut u)
      It is the number of the message (and then it's the value of the constant also).
      If you leave it at -1 the next free number will be used.
      NOTE: You should not add change this number while you're just translating a catalog. This option is just meaningful if you are creating or editing a catalog file of a program of yours.
    • Elements
      This field just shows the number of messages in the catalog.
    • Orig
      This filed shows the original message, before any modification you made.
    • New (Shortcut n)
      This is the field where you can change/edit the catalog's message.
    • Base
      This field shows the real original message. It rappresents the first message ever appeared and cannot be changed in any way.
    • Const (Shortcut c)
      This field shows the constant name of the message.
      TIP: If you have just imported a catalog file inside ReCatItPro and not a Catalog Description or a Workspace, then this Constant Name have been generated automatically starting from the text of the message, according to the rules you have in the Vocabulary.dat file.
