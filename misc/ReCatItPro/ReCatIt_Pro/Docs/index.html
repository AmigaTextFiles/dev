ReCatIt Professional

Introduction

ReCatIt Professional, or ReCatItPro for short, is a brand new tool that help developers and localizers to edit and create .catalogs files.
But it isn't just that. ReCatItPro is an all-round tool that will assist you in all production steps, starting from the creation of your first Catalog Descriptor (.cd) file through the production of the final Catalog (.catalog) file.
Even more important, ReCatItPro is also able to perform tasks that usually don't belong strictly to a .catalog editor, but that ease your task even more. For example, it can extract all messages embedded inside a source file, creating the relevant .cd file and also an ASCII file containing all the constant names (created on the fly by ReCatItPro) related to your messages. Since ReCatItPro is a tool born to help developers, it can create source code for many languages: C/C++, Amiga E, Blitz Basic and ASM.
ReCatItPro introduces also the Workspace concept, an environment that keeps track of a lot of things that the localizer should remember.
Since ReCatItPro was born from ReCatIt users, especially ATO ones, it tries to match the requests and the needs of everyday use, giving you the right power and features you need in every day usage.
ReCatItPro is very easy and handy to use, but I suggest you to read all the docs to manage it in its whole power.
