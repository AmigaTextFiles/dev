For texts, you can go to Game Setup screen, and browse the language files 
using the language button. Then edit each of them keeping the text order.

For now only Engligh, French, German, Italian, Spanish, Portuguese and Dutch
are supported.

Images can be localized too, just create the same image with a suffix at
the end end of the name.

Being the suffix one of these:

          French ".FR"
          Italian ".IT"
          German ".DE"
          Spanish ".ES"
          Portuguese ".PT"
          Dutch ".NL"
		  
For example, if you have a background file named menu.iff, and you add Italian 
texts, you can additionally create a menu.iff.IT file. If Italian language is
selected this menu.iff.IT image will be loaded instead of menu.iff.
