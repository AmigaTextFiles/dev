Align to Grid: Sets the object aligned with the closest tile position
For example, if tile size is 16, valid positions for the object will be 0, 16, 32, 48, etc.
Usage: Align to Grid <Axis> <Round off>
Without round off a tile in position 10 will be moved to position 0, but with round off it will be moved to position 16.
