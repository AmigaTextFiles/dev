ITD * Image To Disk

Tool per scrivere immagini disco in un floppy.

Installazione

ITD è contenuto solo nell'archivio DEV, non in quello USR
copialo in C: o in un posto simile

Opzioni

FILE/A,DEVICE,FORMAT/S
FILE/A nome dell'immagine da scrivere
DEVICE il device in cui ITD dovrà scrivere il disco
default "DF0:"
esempio "RAD:"
FORMAT/S   se impostata il disco verrà formattato

Return code (codice di ritorno)

Impostato a 0 se è andato tutto bene, altrimenti a 20

Background

ITD può scrivere in ogni device di tipo trackdisk. Per evitare problemi (ad esempio corrompere i dati nel tuo harddisk) per il momento non scriverà in device più grandi di 2.000.000 di byte. NON ci sarà nessuna verifica sui dati scritti (è solo un programmino scritto in fretta ;), sei stato avvertito! Se l'immagine è più piccola del device ne verrai avvisato, in questo caso ITD non modificherà le tracce rimanenti. Anche nel caso in cui l'immagine fosse più grande verrai avvertito, e ITD scriverà fino alla fine dello spazio disponibile sul device.
