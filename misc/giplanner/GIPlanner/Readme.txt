INTRODUCTION


GI Planner is a tool aimed at Project Managers.
The purpose of the program is to plan the time and costs of a project.
It's a port of my GI Planner program for Windows
(http://www.iacosoft.com/home/programmi2.asp#PLANNER) with which you can
share the produced files.

USE
To launch it, just double click on its icon.
The software was written in C ++ and requires no installation.
The software, once launched, will open a window.
