The History of AutoDocArt Project:

0.01a	Init version.
		Displays directory of autodocs with NListtree MCC.
		Displays document with NFloattext MCC.
		Makes syntax some reserved word (NAME,FUNCTION & etc.)
		Works from cli (with one argument - AUTODOCS/K).

0.02a	First-bugs-fix version.
		Use pattern #?.doc
		Another images.
		Works from Workbench too (with one tooltype - AUTODOCS).
		Faster startup.
		More reserved words (25 at all).

0.03a	Internal new version.
		Total recompilation (OOP reconstruct for future).
		Some images changing.
		More reserved words.

1.00b	First Beta version.
		Some images changing.
		Added prefs window, about window, MUI settings.
		Added Locale support (Russian catalog).

1.01a	Syntax Prefs support.
		Some bugs fixed.

1.020	Made more standart Prefs window.
		Added support ENV: and ENVARC: vars.
		Used new Document-Table-Scan algoritm. TABLE OF CONTENTS no need.
		Faster scan and syntax algoritms and data structures.
		Some bugs fixed.

		Support multi assigns (thanks AmiS).

		Some optimization and changes in scan.
		GUI have changed a little. (and images)
		Made history file.

		Simple search engine added (at last).
		All tooltypes and args are removed.
		Some serious bugs fixed.

		Smarter document scan method (a little faster).
