
-----------------------------------------------

To start the Beast system;

Change the directory (CD) to the BeastV1 directory.
Then execute the BST_BuildBeast script.

	CD <parent directory>BeastV1
	Execute BST_BuildBeast


-----------------------------------------------

To end the Beast system;

Change the directory (CD) to the BeastV1 directory.
Then execute the BST_KillBeast script.

	CD <parent directory>BeastV1
	Execute BST_KillBeast


-----------------------------------------------
