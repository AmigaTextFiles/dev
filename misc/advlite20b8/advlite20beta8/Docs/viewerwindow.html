The Viewer Window

The viewer window displays the actual text of a function. Also, it's possible to jump another function in the same autodoc or to an different autodoc in another group. If the user has enabled the usage of includes, it's also possible to jump to an include file.


Snapshot of Viewer window...

The user are able to select color and font style of the keywords and matching symbols found in the function text. The snapshot above has the keywords in bold blue font and the symbols underlined in white.

When a symbol to an include are selected an window called include window pops up, containing the include file, where the topmost line (if possible) is the matching symbol the user have selected.

There are also some hotkeys the might be handy sometimes.

  • Arrow key up or down, scroll upwards or downwards in the function text
  • Arrow key up or down while holding `LSHIFT` or `RSHIFT` down, to jump one page up or down
  • Arrow key up or down while holding `CTRL` down, to jump to the end or top of the function text
  • Tabulator for activate the symbol window if opened, else it will activate the include window if the include window is not opened the group window will be activated


Image buttons

The image buttons purpose is to make AutodocViewer Lite v2.0 easier to use. These button can be disabled or in other word removed from the display. See the AutodocViewer Settings window for more information.

  Search for a string

The user can search in the current viewed function for a given string. If the user wants to do a cross autodoc search he/she can do that by selecting "Search -> Search..." in the menu.

This function can be selected from

  • This fast image button, if enabled
  • The screen's menustrip


  Search for next string

This will simply search for the next matching string in the function.

This function can be selected from

  • This fast image button, if enabled
  • The screen's menustrip


  Rescan the function for symbols

By selecting this button, AutodocViewer will rescan the currently viewed function for symbols. Usefull if the user have added a new autodoc or updated includes.

This function can be selected from

  • This fast image button, if enabled
  • The screen's menustrip


  Open the viewer options window

This button brings up the viewer options window, were the user can change the apperance of the viewer window.


Snapshot of Viewer Options window...

In the Viewer options window you can select the color of the keyword and links found in the function text. Also, you can change to font styles, into bold, underlined or italic.

This function can be selected from

  • This fast image button, if enabled
  • The screen's menustrip
