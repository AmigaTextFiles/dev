The Settings Window

The Settings window contains many user defined variables, the user may change some features to suite their needs.


Snapshot of Settings window...

Alphabetical Group Window

By ticking this checkbox, AutodocViewer Lite will automatically alphabetize the groups if not, the user may change the order of the groups as preferred.


Toolbar

By ticking this checkbox, the windows toolbars will be displayed


Assembly Includes

This check box indicates wheter AutodocViewer Lite should search for assembly includes or c includes. This feature are not available yet!!! AutodocViewer Lite only supports assembly includes for now.


Include Path

This string shows the include path, it's currently not implemented


Image Path

This string indicates the path to images to use as toolbar images, default path is PROGDIR:images.


Maximum Chronicle

This number indicates how many steps that AutodocViewer Lite should remember. This value is assosiated with the Chronicle window. Default are set to 25.


Bookmark Autosave Timeout

An value in seconds which indicates how often the last viewed function should be bookmarked. The value are stored in the AutodocViewer's PROGDIR: in an file called snapshotlast. Very handy when your current project crashed :-)

Vanilla Timeout

Value indicating keystroke timeouts, this value is currently not used, and will probably be removed.
