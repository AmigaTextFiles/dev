The Autodoc Window

The Autodoc window shows all the autodocs attached to a single group. Any number of autodocs can be attached to a group, but, AutodocViewer will not allow an autodoc to be added twice within the same group.


Snapshot of Autodoc window...

There are also some hotkeys the might be handy sometimes.

  • Arrow key up or down, to select previous or next autodoc
  • Arrow key up or down while holding `LSHIFT` or `RSHIFT` down, to jump one page up or down
  • Arrow key up or down while holding `CTRL` down, to jump to the end or top of the autodoc list
  • Tabulator for activate the function window


Image buttons

The image buttons purpose is to make AutodocViewer Lite v2.0 easier to use. These button can be disabled or in other word removed from the display. See the AutodocViewer Settings window for more information.

  Adds a autodoc

With this button you can add one or more autodocs to the selected group, if the autodoc already exists in the list of autodocs, you will be prompt with a requester asking you wether you want to skip the current autodoc or overwrite the existing autodoc, in other words delete it.


Snapshot of Adding Autodoc window...

Also, if an autodoc file contain error, eg.

  • Table Of Contents was no found...
  • The Table Of Contents entries does not match the function entries...
  • The Table Of Contents contains errors...
  • The autodoc has no ASCII code (0x0C) formfeed...
  • The autodoc has too few or to many ASCII code (0x0C) formfeeds...
AutodocViewerLite can try to correct these errors, but take care this feature are not 100% relayable, thus, it has never given me problems. Note that the autodoc that are to corrected are backed up.

This function can be selected from

  • This fast image button, if enabled
  • The screen's menustrip
  • The hotkey `RightAmiga + a`


  Delete the selected autodoc

By selecting this button you can remove the selected autodoc from the autodoc list, you will be ask if really want to delete it or not.

You can select this function from

  • This fast image button, if enabled
  • The screen's menustrip
  • The hotkey `RightAmiga + d`
  • When the autodoc window is active you can also use the "Del" key


  Move the selected autodoc upward in the autodoc list

Rearrange the order of the autodoc, this button will send the selected autodoc one step upwards in the autodoc list. This button are ghosted if you have selected the alphabetical option in the Groups options window, or if the selected autodoc is the on top of the autodoc list.


  Move the selected autodoc downward in the autodoc list

Rearrange the order of the autodoc, this button will send the selected autodoc one step downwards in the autodoc list. This button are ghosted if you have selected the alphabetical option in the Groups options window, or if the selected autodoc is on the bottom of the autodoc list.


  Sort the whole autodoc list alphabetically

This button will rearrange the whole autodoc list in alphabetically order, the button will be ghosted if you have selected the alphabetical option in the Group options window.
