NOTE: Avant d'installer les fichiers développeurs, vous devez déjà
      avoir installé l'archive gms_user.lha sur votre disque dur.


1. Vous pouvez copier le répertoire GMSDev/ là où vous voulez mais
   nous vous recommandons de le mettre dans GMS: pour pouvoir y
   accéder plus vite.

2. Editez votre fichier S:user-startup et ajoutez les lignes suivantes:

     Assign GMSDev:   <Mettez ici l'emplacement de GMSDev, eg GMS:GMSDev/>
     Assign INCLUDES: GMSDev:Includes/ ADD

   Si vous prévoyez d'utiliser le langage E, ajoutez aussi:

     Assign EModules: GMSDev:EModules/ ADD

3. Si vous avez CygnusEd et êtes intéressé par les sources assembleur,
   copiez Extras/ceddefaults.s dans votre répertoire S:. Cela va définir
   correctement les tabulations des fichiers sources (qui ont une 
   extension .s).
