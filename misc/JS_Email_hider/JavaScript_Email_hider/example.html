It works by listing the ASCII codes of the text to hide, assuming that the
spam spider is trawling web pages for a standard looking email address such as
spam_me@junkmail.com to add to it's database.
Unless it's smart, the spider will not see your email address as it's not actually
written in your HTML page as it would expect to find it.
The JavaScript should work fine in any semi-modern web browers, including IBrowse 2.x,
Voyager 3.x and AWeb 3.x.

If you don't see the two JavaScript MAILTO: links on this page, you may be using a browser
which does not support JavaScript, or you have Javascript disabled.
