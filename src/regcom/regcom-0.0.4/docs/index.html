Background

Regimental Command is the name of a project started in 1997 in order to assist with record keeping for large BattleTech campaigns. The focus eventually turned toward the development of a generic armored combat system. A rough prototype of a map scrolling engine was written using MFC and DirectX and this prototype was then ported over to the Simple Direct Media Layer, a free cross-platform multi-media development API.
