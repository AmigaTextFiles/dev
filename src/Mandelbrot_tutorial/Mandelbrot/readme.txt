Mandelbrot

This is a simple mandelbrot fractal generator with resizable window.

Autor: Norman Walter

Usage:

Keyboard: +/- zoom in/out
Cursor keys: Move up/down, left/right

I made this program as a tutorial source for my C-workshop 
(see http://www.norman-interactive.com/C-Kurs.html).

It shows the usage of pen sharing via ObtainBestPen() and how to
handle Intuition-messages.
