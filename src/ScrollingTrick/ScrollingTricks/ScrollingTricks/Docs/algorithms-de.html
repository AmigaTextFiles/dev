Die hier beschriebenen Algorithmen und die Demo Source Codes verwenden alle der Einfachheit halber und zur Kompatibilität mit allen Versionen des Amiga Chipsets (OCS, ECS, AGA):
  • eine Blockgröße von 16 x 16 Pixel
  • eine Bitplane Tiefe von 4 (d. h. 16 Farben)
  • eine sichtbare Bildschirmfläche von 320 x 256 Pixel
  • Single-Buffering
  • eine Bitmap im interleaved Format

Bevor Sie sich die Dokumentation zu den einzelnen Algorithmen anschauen, lesen Sie unbedingt die allgemeinen Hinweise, die für alle Scroll Techniken gültig sind!
