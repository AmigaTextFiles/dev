Latest version

Join the portrait mailing list to receive information about the latest version.
The last publicly release version of Portrait was 1.3

The version currently in development is 1.4

This version will incorporate the following new features:

  • Much more stable (hopefully)
  • New virtual memory system
  • New draw modes (eg. clone)
  • Better integration in the Colour window

The entire source has been rewritten from scratch (or is in the process of). This means V1.4 should be a lot more stable and it will make it easier for me to add features, etc. later on.
