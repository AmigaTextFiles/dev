Rendered images


The 3d shapes/models used to render these images with Reflections 4 ,also known as monzoom, were completly made with the TDO library, using the Reflections output.

This pawn was made using the rotation functions of the library. I put manually a mathematical texture on it in Reflections.
This mathematical function was made using the ripples formula out of Plotter 3D by Sven Steiniger.
The famous Tron tank, was modelled by Stephan Bodmer and can be found in his tools, like GEO2VRML or VRMLViewer and Editor.
The tank as Videoscape GEO file was converted with the library by using its standart loader and saver modules.
A fractal landscape, using the source supported by the Amiga-Magazin their OpenGL workshop, and modified to work with the library.
A shell, an Ammonite, generated with the modified source of Shelly V1.6M, found on a MeatingPearls CD-ROM, by Randolf Schultz.
This mathematical function was made using the wave formula out of Plotter 3D by Sven Steiniger.
