Requirements

The library and its modular components made by the author will run on any 68k based Amiga with OS3.0.

It is well known that 3D uses a lot of memory. I tried to keep the usage as low as possible. But to speed up things memory will be allocated in blocks. This means that for a simple cube with 8 vertices and 6 faces will use nearby the same amount of memory as a mesh uses which has some hundred vertices and polygons. Finally, the size of a mesh you want to generate depends only of the amount of memory installed in your system.
