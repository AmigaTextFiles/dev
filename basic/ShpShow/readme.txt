Usage:
------

START:

  Only from CLI: 

     ShpShow IFF-file shapes-file

  IFF-file needed for the definition of the screen, where the shapes
           are displayed.(width,height,palette)

  shapes-file the shapes 


CONTROL:

  Cursor up+down   next/prev shape

  Cursor left+right  next/prev color for shape-number (If the first colors are equal.)

  ESC  end


That's it!  A little, but very usefull tool. :-)
