Overview
--------

This package provides the following files:

- COMMANDLINE.LST
- CONPRINT.LST
- systemtaglist.LST

COMMANDLINE.LST
...............

The @commandline$ function returns the command line passed to your program.
(You should not use this function.)

To get the command line arguments passed to your program, do the
following:

  (1) DIMension the arg$() array at the top of your program
  (2) Change the value "127" of the @numargs function to the value used
      in step #1
  (3) assign the result of @numargs to a variable
  
  Result: arg$() holds the command line arguments
          the variable used in step #3 holds the # of arguments.
          
CONPRINT.LST
............

The @conprint() command prints a text string to the current CLI window.

   (1) Alter the @conprint() sub program if you want to use another file #
   (2) Use @conprint("Text") to print "Text"
   (3) Use @conprint("") to print a blank line

CLI input is analogous to CLI output. You must use the "*" file to achieve
this.

systemtaglist.LST
.................

The @systemtaglist() command executes an external program.

   (1) Use @systemtaglist("list C:") to execute the "list c:" command.

Überblick
---------
                                   
Die Programme bieten die folgenden Funktionen:

  Programm           Funktion(en)
  -------------------------------------------------------------------
  COMMANDLINE.LST    Die interne Funktion @commandline$ gibt einen
                     String mit allen Argumenten zurück.
                       Nach der Dimensionierung des Arrays und der
                     Anpassung des Wertes 127 in der Funktion sollten
                     Sie einer Variablen das Ergebnis der Funktion
                     @numargs zuweisen. Die Variable enthält danach
                     die Anzahl der Argumente; arg$() enthält die
                     einzelnen Argumente.
  CONPRINT.LST       Rufen Sie @conprint("Hallo") auf, um diesen
                     Text im CLI-Fenster auszugeben. @conprint("")
                     gibt eine Leerzeile aus.
                       Hinweis: Passen Sie ggf. die Dateinummer an.
  systemtaglist.LST  Benutzen Sie @systemtaglist("list"), um den
                     Inhalt des aktuellen Verzeichnisses anzuzeigen.
  
Die Eingabe über das CLI muss analog zu Textausgabe ebenfalls über die
Datei "*" erfolgen.
