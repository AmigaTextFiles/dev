Welcome!

This is a demonstration of the HTML browser written in Blitz Basic for Amiga Format. This is a simple file which contains little in the way of HTML codes. You should be able to add support for all the HTML tags you need without too much effort

Tags which you should consider supporting include list and table tags. You'll also need to deal with links, but I think I have that one already sussed out and we can see if I get working in time for next month.

*&* Having the opportunity to parse the document twice actually makes things a lot easier. If gives you the chance to check graphics files and prepare any images which need rendered. Of course, lack of support for GIF or JPEG in Blitz makes that quite tricky, although a little bird tells me that there could be Datatype support before very long.
