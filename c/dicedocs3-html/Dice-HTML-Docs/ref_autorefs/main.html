                              AutoRefs

 AUTOREFS outfile docfile docfile ...


 AUTOREFS creates a DME reference file from the specified document
 files.  AmigaDOS wildcards may be used to specify the doc files.

 The output file is generally specified as DME.REFS and placed in the
 same directory as the documents it references.  A path to this
 directory is normally created via the ADDPATH command from DME's
 startup script (S:.EDRC).

 The output file is appended to.  For an example, refer to the
 DCC3:MAN/MKREFS script used to create a DME.REFS for the manual pages.
