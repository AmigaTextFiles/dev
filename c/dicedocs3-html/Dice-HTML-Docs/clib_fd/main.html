    Contents: File Pointer Operations

    File Operations
      What is a file_descriptor ?
      open - open file decriptor
      read - read from file decriptor
      close - close file decriptor
      write - write to file decriptor
      creat - create file decriptor (Obsolete)
      unlink - delete file decriptor
      rmdir - remove dir
      mkdir - make dir

    Other File Operations
      fcntl - get file decriptor controls
      fdtofh - get AmigaDOS file handle
      ioctl - get file decriptor I/O controls
      isatty - is decriptor TTY
      lseek - seek within a file decriptor
