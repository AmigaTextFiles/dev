                         Operater Precedence

    -----------------------------------------------------------------
    ()  {}  ->  .                                       Left-to-right
    !   ~   ++  --  +  -  *  &  (type)  sizeof()        Right-to-Left *
    *   /   %                                           Left-to-right
    +   -                                               Left-to-right
    <<  >>                                              Left-to-right
    <   <=  >   >=                                      Left-to-right
    ==  !=                                              Left-to-right
    &                                                   Left-to-right
    ^                                                   Left-to-right
    |                                                   Left-to-right

    ||                                                  Left-to-right
    ?:                                                  Left-to-right
    =   +=  -=  *=  /=  %=  &=  ^=  |=  <<=  >>=        Right-to-left **
    ,                                                   Left-to-right
    -----------------------------------------------------------------
