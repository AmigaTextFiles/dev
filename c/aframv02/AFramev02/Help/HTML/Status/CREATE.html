AFTab/Create

  NAME void = Create(AFWindow *window, AFRect *rect, TabLabel tlabels[], ULONG gadid);


  DESCRIPTION
    This function creates and adds the Tab gadget to the window specified


  INPUTS
    window - AFWindow pointer to which the tab.gadget will be added
    rect - Coordinates and size of the tab gadget
    tlabels - Array of labels
    gadid - Gadget id


  RESULT
    none


  BUGS
    none known
