AFTextDT/SetBuffer

  BOOL = SetBuffer(UBYTE *buffer, ULONG size)


  DESCRIPTION
    Sets the buffer for the current datatype object


  INPUTS
    buffer - UBYTE pointer to text data
    size   - size of text data


  RESULT
    TRUE - if success
    FALSE - if failure
