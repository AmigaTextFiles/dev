TextOut

AFRastPort/TextOut

  void = TextOut(ULONG x, ULONG y, char *text, ULONG length)


  DESCRIPTION
    Allows you to print text out at x,y coordinates in your window


  INPUTS
    x - horizontal coordinates
    y - vertical coordinates
    text - pointer to text to be output
    length - length of text


  RESULT
    none


  BUGS
    none known


  SEE ALSO
