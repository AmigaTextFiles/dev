AFFrameIClass/Create

  void = Create(AFWindow *window, AFRect *rect, int recessed, int frametype)


  DESCRIPTION
    This member allows you to add a frame to a window (raised or recessed)
    with size and position of rect.


  INPUTS
    window    - pointer to an AFWindow
    rect      - pointer to an AFRect
    recessed  - recessed value (i.e. AFFrameIClass::recessedUp)
    frametype - frame type (i.e. AFFrameIClass::buttonFrame)


  RESULT
    none


  BUGS
    none known
