AFTapeDeck/Create

  NAME void = Create(AFWindow *window, AFRect *rect, ULONG id)


  DESCRIPTION
    This function adds the TapeDeck BOOPSI object to a window


  INPUTS
    window - window to which object is to be added
    rect - coordinates and size of object
    id - gadget id of object (not yet supported)


  RESULT
    none
