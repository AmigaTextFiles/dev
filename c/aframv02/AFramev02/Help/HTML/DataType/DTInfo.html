DTInfo

AFDataType/DTInfo

  BOOL = DTInfo()
  BOOL = DTInfo(char *file)


  DESCRIPTION
    This member can be called to get information on an existing object or file.
    The m_dtGlobal.dtInfo structure is filled for your usage if successful.


  INPUTS
    file - pointer to a file name


  RESULT
    TRUE - if success
    FALSE - if failure


  BUGS
    none known
