AFAppWindow/OnCloseWindow

  NAME OnCloseWindow(LPIntuiMessage imess)


  DESCRIPTION
    This function overrides the default AFWindow classes OnCloseWindow in order to
    properly remove the AppWindow from the system and calles the windows default
    DestroyWindow() member.


  INPUTS
    imess - LPIntuiMessage structure


  RESULT
    none


  BUGS
    none known
