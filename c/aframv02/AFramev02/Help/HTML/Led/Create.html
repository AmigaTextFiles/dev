AFLed/Create

  NAME Create(AFWindow *window, AFRect *rect, ULONG id)


  DESCRIPTION
    This funtction adds the Led BOOPSI object to a window


  INPUTS
    window - window to which object is to be added
    rect - coordinates and size of object
    id - gadget id of object (not yet supported)


  RESULT
    TRUE if successful
    FALSE if failure


  BUGS
    none known
