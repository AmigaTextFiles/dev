Installation

   To install reqtools.library copy it to your Libs: directory by hand, or run the installation script 'Install' (by
using 'Execute' from the Shell or by double-clicking its icon).

   If you are running Workbench 2.1 or higher you will have the possiblity to use ReqTools in different languages.
Just use the appropriate install script in the Install/Workbench_2.1+ directory.
