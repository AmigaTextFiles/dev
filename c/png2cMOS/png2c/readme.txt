png2c
usage: png2c infile outfile. [oufile.h]
png2c -v for version info

This is the MorphOS-Version of the small tool to convert
png-Images to ANSI-C sorce code Bitmaps,
wich maybe used to integrate an a PNG-picture into
your project.
It also can convert PNG-pictures with Alpha-channel to
ARGB LONG word bitmaps.
