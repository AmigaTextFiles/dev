Name
pitem:Set -- set item data
Synopsis
pitem:Set(table)
Function
Sets data for the item. The data to set must be passed as one or more key-value pairs in the table argument. The following keys are currently supported:

Offset
Byte offset of the start of this item in text.
Length
Length of this item in bytes.
NumChars
Number of Unicode characters in the item.
Analysis
Analysis results for the item. This is a Pango analysis object.

Inputs
table
a table consisting of key-value pairs of the data to set
