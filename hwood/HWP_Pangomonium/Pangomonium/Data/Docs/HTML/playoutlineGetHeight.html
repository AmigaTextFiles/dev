Name
playoutline:GetHeight -- get line height
Synopsis
height = playoutline:GetHeight()
Function
Computes the height of the line, as the maximum of the heights of fonts used in this line.

Note that the actual baseline-to-baseline distance between lines of text is influenced by other factors, such as playout:SetSpacing() and playout:SetLineSpacing().

Inputs
none

Results
height
return location for the line height
