Installation

APPBuilder does not need any installation procedure, you have to extract the zipped archive package into a folder of your choice.
Once you have extracted the contents you should find in the destination folder the following items:

  • APPBuilder (APPBuilder.exe for Windows version)
    This is the main program, the one you have to execute to start the application
  • config
    This file is used by APPBuilder to store its configuration, in most recent version this file is saved in the AppDir of your system.
  • licenses
    This folder holds all plugins licenses
  • plugins
    Here are all the available Hollywood plugins (not included with the application but downloadable)
  • projects
    A pre-created folder where you can store your projects
  • docs
    A folder with the documentation in both HTML and pdf formats.
  • rraf.def
    This file is needed to grant you the remote server access to download the available plugins
  • background.png
    An image file used as splash screen during the application loading
