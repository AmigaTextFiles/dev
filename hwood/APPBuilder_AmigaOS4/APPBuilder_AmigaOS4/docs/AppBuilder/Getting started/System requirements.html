System requirements

Here are the requirements:

  • Hollywood : This is obvious if you want to build Hollywood projects, the latest version has been tested with Hollywood 10, it may not work correctly in the previous Hollywood version.
  • Screen resolution : 800x600 or greater
  • 7zip installed on your system : This is needed if you want to pack your application builds in zipped archives
  • If you are going to run the application on a real Amiga you will also need:
    • RTG System
    • Some additional fast memory (>8 Mb)



That's all.
