Project management

Let's see what's in the left buttons column.

The top left buttons column has all you need to manage your projects:

This group, called “Project” features the following buttons:

- LOAD : Click it to load a previously created project
- SAVE : Click it to save the current project using the same file name if it's an existing project or the program will ask you for a new project file name.
- SAVE AS : Click it to save the current project with another name
- CLEAN : Click this button to clean the interface ant to clear all the project's fields, switches & resources.


 

 

The middle group is called “Switches” and allow you to enable/disable some compilation flags along with some other useful options:

- COMPRESSION : If enabled the built executable will be compressed (Hollywood inbuilt feature)
- PACK : If enabled the program will produces for you a zip archive with the built application, one for each target platform. Note that this function will use the 7zip executable you have specified in the configuration window.
- CONSOLE MODE : If enabled Hollywood will build the executable as a console program, it is useful only for Windows targets and it is a new flag introduced with Hollywood 10.
- LOCK SETTINGS : If enabled the compiled application will have a lock that not allow the final user to use some flags when running the application from a terminal or from a shell.
 


 

The third group holds the following buttons:

- CHECK FOR UPDATE : It's used to let you know if a new version is available and to update the plugins to their latest version.
- CONFIGURE : We have seen it earlier, it is used to cofigure the Appbuilder program.
- ABOUT : Display a window with some informations about the program along with the registration informations.
- WEBSITE : Opens your browser and points to the AppBuild web page.
- BUILD : Starts the build process and produces the build packages, one for each selected target platform.

 

 

 

The third section holds three buttons as showed below:


CONFIGURE
        Open the configuration window.
ABOUT
        Open a window with informations about APPBuilder.
WEBSITE & DOCS
        Will open your browser to the web page of this application.


The last button you have is the BUILD button:


Clicking this button will start the build process. A status window will be opened so you will be able to view what's happening.
Have a look at the tutorial for a complete explanation.
