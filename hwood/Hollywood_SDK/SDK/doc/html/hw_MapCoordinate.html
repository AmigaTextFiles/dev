Name
hw_MapCoordinate -- map special coordinate (V9.1)
Synopsis
int v = hw_MapCoordinate(int horiz, int c, int size, int parentsize,
            double *anchor, struct hwTagList *tags);
Function
This function can be used to map one of Hollywood's special coordinates like #CENTER, #RIGHT, #BOTTOM, etc. to an absolute coordinate. The horiz parameter has to be set to True or False, indicating whether the coordinate is in horizontal or vertical view space. The c parameter must be set to the special coordinate to convert. size must be the size of the reference object and parentsize must be the parent's view size. In case horiz is False, size and parentsize must be set to a height value and if horiz is True, they must be set to a width value. The anchor parameter is optional and can be set to a pointer to a double containing the reference object's anchor point. Alternatively, it can be NULL.

Designer compatibility
Unsupported

Inputs
horiz
True for horizontal, False for vertical view space
c
special Hollywood coordinate to map
size
size of the reference object (width or height, depending on mode)
parentsize
size of the reference object's parent container (width or height, depending on mode)
anchor
pointer to a double containing the object's anchor point or NULL
tags
reserved for future use; pass NULL
Results
v
converted coordinate
