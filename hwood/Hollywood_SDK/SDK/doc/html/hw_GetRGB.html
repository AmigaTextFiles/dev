Name
hw_GetRGB -- convert pixel format to RGB (V6.0)
Synopsis
ULONG rgb = hw_GetRGB(ULONG color, int infmt);
Function
This function converts the specified color from the specified pixel format to RGB. See Pixel format information for details.

To convert a color from RGB to an arbitrary pixel format, use hw_MapRGB(). See hw_MapRGB for details.

Designer compatibility
Unsupported

Inputs
color
color to convert
infmt
pixel format of source color
Results
rgb
converted color
