Project description

In the top right section you can find some string fields where you can select the main source code and you can describe the project, let's see all available fields:

- PROJECT NAME : Type here your project's name
- MAIN SOURCE : Type or select with “…” button the main soruce code of your application
- EXECUTABLE NAME : This field is used to define the executable's name, but actually has two purposes
  - The path of the selected file will be used as build folder
  - The file name will be used as the executable file name, for Windows executables the .exe suffix will be added automatically.
- .INFO ICON : For Amiga-like targets you can also select an .info icon that will be copied along with the executable.
- NOTES : A multi-line text field where you can write your personal notes about the project.
