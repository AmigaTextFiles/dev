Name
page:SetCMYKStroke -- set CMYK stroking color
Synopsis
status = page:SetCMYKStroke(c, m, y, k)
Function
page:SetCMYKStroke() sets the stroking color. The individual parameters must all be between 0 and 1.

Inputs
c
level of cyan
m
level of magenta
y
level of yellow
k
level of black
Results
status
status code
