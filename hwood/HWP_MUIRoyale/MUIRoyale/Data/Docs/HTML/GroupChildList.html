Name
Group.ChildList -- get a list of all group children (V1.2)
Function
This attribute allows you to get a table that contains information about all children that are currently attached the group. For each child in the group, the following table elements will be initialized:

ID:
Contains the ID of the group child.

Class:
Contains the name of the class this object belongs to, e.g. "Cycle".

MUIClass:
Contains the internal MUI class name for this object, e.g. "Cycle.mui". This is probably of not much interest for your application.

Type
Table

Applicability
G

Example
t = mui.Get("mygroup", "childlist")
For Local k = 0 To ListItems(t) - 1
   DebugPrint(t[k].id, t[k].class, t[k].muiclass)
Next
The code above dumps all objects that belong to "mygroup".
