Name
Area.Hide -- show/hide object
Function
Objects with this attribute set are not displayed. You can set Area.Hide at any time, causing objects to appear and to disappear immediately. A new layout is calculated whenever some objects are shown or hidden. When necessary, MUI will resize the parent window to make place for the new objects.

Currently, MUI does a complete window refresh after showing/hiding objects. This behaviour might get improved in the future.

Type
Boolean

Applicability
ISG
