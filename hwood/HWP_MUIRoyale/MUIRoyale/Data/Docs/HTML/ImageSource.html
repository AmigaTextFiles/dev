Name
Image.Source -- set image to display
Function
Set the image to display. The following predefined MUI images are available here:

  • – ArrowDown
  • – ArrowLeft
  • – ArrowRight
  • – ArrowUp
  • – Assign
  • – Brush:<id>
  • – CheckMark
  • – Chip
  • – Cycle
  • – Disk
  • – Drawer
  • – HardDisk
  • – ListCursor
  • – ListSelCur
  • – ListSelect
  • – Network
  • – PopDrawer
  • – PopFile
  • – PopUp
  • – PropKnob
  • – RadioButton
  • – SliderKnob
  • – TapeDown
  • – TapePause
  • – TapePlay
  • – TapePlayback
  • – TapeRecord
  • – TapeStop
  • – TapeUp
  • – Volume

If you use the type Brush you need to set <id> to the identifier of a Hollywood brush that you want to use for this image object.

Type
