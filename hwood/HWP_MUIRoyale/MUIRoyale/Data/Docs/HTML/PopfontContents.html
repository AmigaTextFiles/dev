Name
Popfont.Contents -- set/get string gadget contents
Function
Get and set a string gadget's contents. Popfont class uses a slash to separate font name and size, e.g. topaz/8.

Popfont.Contents gets updated every time when the contents of the string gadget change. When you set up a notification on this attribute, you will hear about every keystroke.

Type
String
