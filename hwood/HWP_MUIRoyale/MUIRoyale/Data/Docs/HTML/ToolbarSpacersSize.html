Name
Toolbar.SpacersSize -- define spacer size
Function
Defines the used size of spacers relative to the size of a button. The following values are possible:

Quarter
Quarter of a button. This is the default.

Half
Half of a button.

One
Same as a button.

This overwrites the user preferences and so must be used wisely. If in doubt, don't use it.

Type
String (see above for possible values)

Applicability
ISG
