Name
Texteditor.SetUnderline -- toggle underline style of text block
Synopsis
mui.DoMethod(id, "SetUnderline", startx, starty, stopx, stopy, flag)
Function
This method toggles underline style on a text block that is defined by four coordinates (startx, starty, stopx, stopy). If the flag argument is set to True, underline style will be added to the specified text block, otherwise underline style will be removed from the text block.

Startx, starty, stopx, and stopy can either be absolute coordinates or one of the following special values:

Min
Specifies the first line or column.

Max
Specifies the last line or column.

Inputs
id
id of the text editor object
start_x
start x position
start_y
start y position
stop_x
stop x position
stop_y
stop y position
flag
boolean flag that indicates toggle state
