First run

When you execute for the first time this application you have to configure it properly or you won't be able to build your projects.

Here is a screenshot of the current version's main screen:


 

The first step is to configure the application clicking on the “Configure” button in the right side, this window will be displayed:

 

Hollywood executable : here you have to provide the full path to the Hollywood executable, you can type it or hit the button “…” on the right to open a file requester and select the file.

7Zip executable : here you have to provide the full path to the 7zip executable, you can type it or use the right button “…” to open a file requester and select the file.

Plugins folder : The last field is used to hold all the available plugins, unless you have your own plugins folder in your system you should leave this field untouched, otherwise type your custom folder or use to button “…” on the right to open a path requester.

“Use the build date in the destination folder” : This switch is enabled by default and is used to create subfolders with your compiled application using the build date & time as part of the folder name, it is very useful to let you know when you have built a certain application.

“Open the destination folder after build” : This switch, if enabled, will open the build folder as soon as the build process has been completed.

When you have completed the configuration click on the “SAVE PREFERENCES” button to write your preferences.

The button “Reset prefs” is used to clear all the file/path fields and to restore the switches to their default values.

Now you are ready to create your first project.
