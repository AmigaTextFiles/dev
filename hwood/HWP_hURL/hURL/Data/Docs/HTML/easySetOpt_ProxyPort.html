Name
easy:SetOpt_ProxyPort -- port number the proxy listens on
Synopsis
easy:SetOpt_ProxyPort(port)
Function
Pass a value with this option to get the proxy port to connect to unless it is specified in the proxy string #CURLOPT_PROXY or uses 443 for https proxies and 1080 for all others as default.

The port number is 16 bit so it can't be larger than 65535.

Inputs
port
