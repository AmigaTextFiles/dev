Name
Menuitem.Help -- set/get menu item help text
Platforms
Windows, Linux, macOS

Function
Set or get the menu item's help text. This text is automatically shown in the status bar of the window that hosts the menu whenever the mouse hovers over the menu item. This is quite convenient for the user because it explains the function of the individual menu items in a somewhat greater detail. See Statusbar class for details.

Type
String
