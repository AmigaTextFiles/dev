Introduction

This is not an official MUI web site. I found that I needed a way of accessing additional programming information on MUI so I built these pages. Feel free to link to this. If you are doing something with MUI and you have it available by HTTP send me the URL and I'll add a link here.

The MUI Mailing List

This is the primary means of obtaining help with MUI problems. Send your postings to:

mui@sunsite.Informatik.RWTH-Aachen.DE

To subscribe or unsubscribe I think you use the address:

mui-request@sunsite.Informatik.RWTH-Aachen.DE

The MUI mailing list is archived here. Note that this archive does seem to miss a lot of messages, that's one of the reasons I set up some of these pages.

To unsubscribe send an empty message to mui-request@sunsite.informatik.rwth-aachen.de with the word unsubscribe in the subject.
