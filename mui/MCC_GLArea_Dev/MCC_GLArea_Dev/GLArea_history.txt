GLArea.mcc MUI Custom Class
(important points only...)

V 0.9.0 (26.12.2000)
-------------------
- Removed image/texture handling and created a new MCC ImageDB.mcc using guigfx.library
  for texture manipulation
- Fixed MUI_Version and MUI_Revision returns
- Removed all specific code for CyberGFX or Picasso96 (no more needed)
- Cleaned up the code in many ways ;^)

V 0.79.1 (14.3.2000)
--------------------
- Added image/texture database handling via ImageManager.library written by Alan Odgaard
- Added different Methods for Images/texture manipulation
- Image scaling is now possible (MUIM_GLArea_ScaleImage)
- Added Standard init function
- Switched from CyberGfx to Picasso96

V 0.61 (31.8.1999)
-----------------
- Fix Cyberguards hits for MUIM_Draw method


V 0.6 (7.8.1999)
----------------
- New rewrite of inter-process communications (no more MsgPorts)
- Two different version, one compiled for 68060/40, the other for 68030/68881
- Semaphore protected shared-variable for different MUI Application accessing the GLArea class
- Debug console is now CON: (instead of KCON:)
- Fixed a datatype error in GLArea_mcc.h include file (APTR -> ULONG)
- Changed MUI tag values (definitive tag values), no more compatible with 0.5.
- Changed stack size of sub-process to 128KB

V 0.5 (30.7.1999)
----------------
- First public Beta release
