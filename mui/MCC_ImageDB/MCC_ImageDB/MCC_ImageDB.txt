MCC_ImageDB 0.1 (26.12.2000)
This version is a Beta stage
Use it at your own risk
Compiled for 68020 or 68040

Introduction:
-------------
This class is a sub-class of MUI_Notify and let you load images via guigfx.library
written by Timm S. Mueller, and store them in the class like a database.
You could then access the image trough a key (name).

The class is usful in conjonction with GLArea.mcc, ImageDB.mcc let you load
an image and convert it io OpenGL/StormMesa textures.

This class is "very young" and could be improved a lot with new functionalities...
but as it is now, it does what I needed ;^)

Requirement:
------------
You need:
-guigfx.library, render.library
-68020+
-MUI 3.x
-Kickstart 2.04+
-Some Fast RAM

Installation:
-------------
Copy ImageDB.mcc (or ImageDB_68020.mc) to MUI:Libs/mui
There is no Global preference for ImageDB.mcc (I don't know how to write a mcp...)
