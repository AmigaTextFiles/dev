Preferences

You may configure HTMLtext class in the way you want. Therefore you start the MUI configuration program and click on the line called "HTMLtext".

The preferences are divided in the following three parts:

  • MISC
    • Load Images: You may switch off image loading to fasten up the document loading times. (Default: ON)
    • Underline links: If you prefer underlined links, choose this option. (Default: OFF)
    • Use default colors: If selected the colors defined in the documents BODY tag will be ignored and the default colors will be used (Default: OFF)
    • Cycling: Enables cycling with the TAB key. Using this mode you may work with HTMLtext.mcc without using the mouse (Default: ON)
    • Marking: Enables text marking. Usefull if you want to transfer text from a HTML text in the clipboard for example. (Default: ON)
  • FONTS
    • Choose fonts to be used with the header and with some text styles. The font specification must have this form: "fontname/size/styles" e.g "times/18/ib". The style part has to be composed of the characters b, i and u, where b stands for bold, i for italic and u for underlined.
      The first seven lines handle the font size of the FONT tag. Where 7 is the biggest, 1 the smallest and 3 the normal font size.
  • DISPLAY
    • Text: Color of the text. (Default: MUI text color)
    • New Links: Color of the new links. (Default: blue)
    • Followed Links: Color of the followed links. (Default: dark blue)
    • BLINK: Color to be used with the BLINK tag, because it is quite difficult to program a blinking font, so this way of marking is prefered. (Default: red)
    • Marking: Color of the background of marked text. (Default: yellow)
    • Margin left/right: Percentage of the displays width to be left free on both sides of the text.
    • Margin top/bottom: The same as above but for the top and the bottom.
