Supported tags

Here is a list of all tags since HTML 3.2 that I know (Taken from Günter Born, HTML 2.0/3.2, Addison-Wesley 1996).

Not supported tags are in brackets. After the tagnames you may find already supported attributes. The plus symbols (+) indicate, that I plan to implement the tags or want to add some attributes or have to modify the existing code. Attributes or tags with a heading asterisc (*) are only supported by HTMLtext.mcc and may not be used, if the text should be readable by other browsers. Some tags are supported, but not in the way they should do, e.g. BLINK doesn't let the text really blink, but changes the color of the font. These tags are marked with an apostrophe (").
