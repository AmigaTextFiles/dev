-- Command: radio ID/K, COMMAND/K, PORT/K, HELP/K, NODE/K, ATTRS/K/M,
          LABEL=LABELS/K/F
     Radio gadget objects are created with this command.  Cycle and
     radio gadgets are essentially identical with the only difference
     being the base object class to create each type of object.  All
     options are identical for these objects.  Refer to the cycle
     command for descriptions of the options.
