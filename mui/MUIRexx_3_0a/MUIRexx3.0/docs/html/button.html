-- Command: button ID/K, COMMAND/K, PORT/K, HELP/K, NODE/K, ICON/K,
          SPEC/K, PICT/K, TRANS/S, ATTRS/K/M, LABEL/K/F
     Button gadget objects are created with this command.  Text,
     button, switch, check, and image gadgets are essentially identical
     with the only differences being the base object class and the
     default appearance/action. All options are identical for these
     objects.  Refer to the text command for descriptions of the
     options.
