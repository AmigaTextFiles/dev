-- Command: quit FORCE/S
     This command will end the `MUIRexx' application, closing windows
     and freeing all associated memory.  Note that if a script fails
     `MUIRexx' may still actually be running.  Use this command to end
     the process by using an inline `ARexx' command (e.g.  issue `rx
     "address [portname] quit"' from a shell).
