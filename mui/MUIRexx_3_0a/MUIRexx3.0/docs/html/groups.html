Groups
======

   The following commands are used for group creation.  Some arguments
can be used after the groups window is opened to change or retrieve
settings.  In the argument descriptions the keywords are followed by a
pair of [] containing indicator letters.  If the letter I is indicated
then the argument is valid during creation.  If the letter S is
indicated then the argument can be set after the groups window is open.
 If the letter G is indicated then the argument can be retrieved.


  group 
  endgroup
