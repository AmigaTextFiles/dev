-- Command: popslider ID/K, COMMAND/K, PORT/K, HELP/K, NODE/K,
          ATTRS/K/M
     Popslider gadget objects are created with this command.  Slider,
     popslider and knob gadgets are essentially identical with the only
     difference being the base object class to create each type of
     object.  Specifically, while unselected the gadget displays the
     current numeric value in a button.  If selected then a slider pops
     up allowing selection of a new value.  All options are identical
     for these objects.  Refer to the slider command for descriptions
     of the options.
