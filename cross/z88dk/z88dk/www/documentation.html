Documentation

Unfortunately, as with so many projects, documentation hasn't until recently been a high enough priority for z88dk. Hopefully however, things are going to change and you'll be able to glean some useful information from these pages.
Library Documentation

A start has been made on documenting the available library functions in z88dk.

  • Standard Library - Documentation of the standard C Library
  • Games Library - Detailing the library which is designed to make cross platform games programming possible
  • Z88 Library - Details of the extra routines which are available only on the z88 platform
  • Z88 Far Malloc - Details of the far memory implementation for the z88

More documentation is planned to fill in the gaps in these files.
Miscellaneous Documentation
  • Application Creation - Details on how to generate emulator runnable files using z88dk
  • Z88dk on Windows - Henk's guide to running z88dk on Windows
  • ZX Spectrum Screen Driver - Details on the Native ZX Spectrum display driver
  • Retargetting z88dk - Details of how to retarget z88dk for a new z80 machine
