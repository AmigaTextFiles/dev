


Installation

The installation is very straightforward and just consists of the following steps:

1. Copy the archive stasmlnk.zip to a new and empty directory dedicated for ST6
Assmembler development.

2. Just unpack the archive

pkunzip easm.zip

This will result in the following directory tree:


easm.exe
elnk.exe
edis.exe
examples\one.asm
examples\two.asm
examples\three.asm

readme.txt
manual.txt
history.txt
